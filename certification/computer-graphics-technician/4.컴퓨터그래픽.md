## 컴퓨터그래픽 일반

### 컴퓨터그래픽 역사

| 세대  | 시기                      | 소자별                   | 기기별 특징                                               |
| ----- | ------------------------- | ------------------------ | --------------------------------------------------------- |
| 1세대 | 1940년대~1950년대 초      | 진공관(Vacuum Tube)      | 에니악(ENIAC, 1946). 대형·고전력. 패치보드 프로그래밍     |
| 2세대 | 1950년대 후반~1960년대 초 | 트랜지스터(Transistor)   | 소형화·저전력. 어셈블리어 사용. IBM 1401 등               |
| 3세대 | 1960년대 중반~1970년대 초 | 집적회로(IC)             | 다중프로그래밍·시분할 처리. 운영체제(OS) 등장. IBM 360 등 |
| 4세대 | 1970년대 중반~현재        | 고밀도집적회로(LSI·VLSI) | 마이크로프로세서·개인용 컴퓨터(PC) 등장. GUI·네트워크     |
| 5세대 | 현재~미래                 | 초고밀도집적회로(ULSI)   | 인공지능·병렬처리·자연어 처리. AI 컴퓨터·뉴럴 네트워크    |

### 좌표계

| 종류                                       | 특징                                                                                                     |
| ------------------------------------------ | -------------------------------------------------------------------------------------------------------- |
| 직교 좌표계(Cartesian Coordinate System)   | X·Y(·Z)축이 서로 직각으로 교차하는 좌표계. 2D/3D 그래픽의 기본 좌표계. 점의 위치를 (x, y, z) 값으로 표현 |
| 극 좌표계(Polar Coordinate System)         | 원점으로부터의 거리(r)와 각도(θ)로 점의 위치를 표현하는 2D 좌표계. 원·호 등 곡선 표현에 적합             |
| 원통 좌표계(Cylindrical Coordinate System) | 극 좌표계를 3D로 확장. 거리(r)·각도(θ)·높이(z)로 표현. 원통형 물체 표현에 적합                           |
| 모델 좌표계(Model Coordinate System)       | 3D 모델 자체의 기준 좌표계. 오브젝트 내부의 상대적 좌표. 월드 좌표계로 변환하여 씬에 배치                |

## 출력장치

### 모니터

| 종류       | 주요 소자                          | 특징                           | 단점                                       |
| ---------- | ---------------------------------- | ------------------------------ | ------------------------------------------ |
| CRT 모니터 | 음극선관(전자빔+형광면)            | 색재현성 우수. 응답 속도 빠름  | 부피·무게 크고 전력 소비 많음. 전자파 발생 |
| LCD 모니터 | 액정(Liquid Crystal) + 백라이트    | 얇고 가볍고 소전력             | 초기 모델은 시야각·응답 속도 열세          |
| PDP 모니터 | 플라즈마(이온화 가스)              | 대형 화면·자발광·광시야각      | 전력 소비 많음. 번인(burn-in) 현상         |
| LED 모니터 | 발광다이오드(백라이트 또는 자발광) | 얇고 전력 소비 적음. 밝고 선명 | OLED 제외 자발광 아님                      |

### 프린터

| 종류              | 충격/비충격 | 주요소자            | 특징                                      | 단점                       |
| ----------------- | ----------- | ------------------- | ----------------------------------------- | -------------------------- |
| 도트매트릭스 방식 | 충격식      | 핀(바늘)+리본       | 카본 복사본 출력 가능. 전표·영수증에 사용 | 소음 크고 인쇄 품질 낮음   |
| 잉크젯 방식       | 비충격식    | 노즐+액체 잉크      | 컬러 인쇄 우수. 가격 저렴. 가정용         | 속도 느리고 잉크 번짐 가능 |
| 열전사 방식       | 비충격식    | 열 헤드+잉크 리본   | 바코드·라벨·영수증에 사용                 | 전용 용지 필요             |
| 염료승화 방식     | 비충격식    | 열 헤드+승화성 염료 | 연속 계조 표현으로 사진 품질에 근접       | 출력 비용 높음             |
| 레이저 방식       | 비충격식    | 레이저 빔+드럼+토너 | 빠르고 선명. 사무용으로 주로 사용         | 초기 구입 비용 높음        |

### 필름 레코더(Film Recorder)

디지털 데이터를 컬러필름에 출력하는 장치.

### 출력 속도 단위

| 단위                      | 설명                                                              |
| ------------------------- | ----------------------------------------------------------------- |
| CPS(Character Per Second) | 초당 출력 문자 수. 도트매트릭스·잉크젯 등 문자 단위 프린터에 사용 |
| LPM(Line Per Minute)      | 분당 출력 라인 수. 라인 프린터에 사용                             |
| PPM(Page Per Minute)      | 분당 출력 페이지 수. 레이저 프린터 등 페이지 단위 프린터에 사용   |

### 디자인 소프트웨어

| 종류               | 지원 포멧                        | 이미지 방식    | 특징                                                                       |
| ------------------ | -------------------------------- | -------------- | -------------------------------------------------------------------------- |
| 포토샵(Ps)         | JPEG·PNG·GIF·TIFF·PSD·BMP·PDF 등 | 비트맵(래스터) | 사진 편집·합성·보정에 특화. 레이어·마스크 등 다양한 편집 기능              |
| 일러스트레이터(Ai) | AI·EPS·SVG·PDF·PNG·JPEG 등       | 벡터           | 로고·일러스트·타이포그래피·인쇄물 제작에 특화. 크기 변경 시 품질 손실 없음 |
| 오토케드(Auto CAD) | DWG·DXF·DWF 등                   | 벡터           | 건축·기계·설계 도면 작업에 특화. 정확한 치수 계산 및 2D/3D 제도            |

### 컴퓨터그래픽 렌더링

**`매핑(Mapping)`**

컬러와 셰이딩 이후 최종적으로 사실감을 높이기 위해 표면에 질감을 표현하는 과정.

2D이미지를 3D오브젝트에 투영시키는 과정으로, **질감전사** 라고도 한다.

| 구분            | 설명                                                                                                    | 특징                                            |
| --------------- | ------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| 이미지 매핑     | 2D 이미지(텍스처)를 3D 오브젝트 표면에 입혀 색상·무늬를 표현하는 기법                                   | 나무·천·피부 등의 질감 표현에 사용. 사실감 향상 |
| 범프(Bump) 매핑 | 법선(Normal) 벡터를 조작하여 빛·그림자로 표면의 요철(입체감)을 표현하는 기법. 실제 형상은 변경하지 않음 | 벽돌·돌·피부 주름 등의 질감 표현에 사용         |

**`반사 기법`**

| 구분                    | 설명                                                                                                    | 특징                                                            |
| ----------------------- | ------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| 광선추적법(Ray Tracing) | 광원의 빛이 오브젝트에 반사·굴절·투과되는 경로를 역방향으로 추적하여 사실적인 빛을 구현하는 렌더링 기법 | 사실적인 반사·굴절·그림자 표현 가능. 연산 시간이 매우 오래 걸림 |
| 리플렉션(Reflection)    | 오브젝트 표면에 주변 환경이 반사되는 효과를 표현하는 기법. 환경 맵(Environment Map)을 활용하기도 함     | 거울·유리·물·금속 표면의 반사 표현에 사용                       |
