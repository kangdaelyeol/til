## 디자인 일반

### 디자인의 4단계 성립과정

| 과정     | 설명                                                      |
| -------- | --------------------------------------------------------- |
| 욕구과정 | 인간의 필요와 욕구를 파악하고 디자인 목표를 설정하는 과정 |
| 조형과정 | 아이디어를 시각화·형태화하는 조형적 표현 과정             |
| 재료과정 | 목적에 맞는 재료를 선택하고 검토하는 과정                 |
| 기술과정 | 선정된 재료를 가공·제작하기 위한 기술을 적용하는 과정     |

### 산업 디자인 영역

산업 디자인은 크게 시각 디자인, 제품 디자인, 환경 디자인으로 구분된다.

### 시각 디자인(Visual Design)

시각 전달(Visual Communication)디자인, 커뮤니케이션 디자인 등으로 쓰이고 있다.

사람과 사람 사이의 기호에 의해 의미를 전달하는 과정이다.

| 기능          | 시각 디자인 종류                 |
| ------------- | -------------------------------- |
| 지시적 기능   | 교통 표지판, 안내 표지, 픽토그램 |
| 설득적 기능   | 광고, 포스터, 홍보물             |
| 상징적 기능   | 심볼 마크, 로고, 캐릭터          |
| 기록적인 기능 | 사진, 다큐멘터리, 영상 기록물    |

**`시각 디자인 분류`**

| 분류                          | 설명                                                                           |
| ----------------------------- | ------------------------------------------------------------------------------ |
| 그래픽 디자인                 | 이미지, 문자, 색상 등을 조합해 시각적 커뮤니케이션을 구성하는 디자인           |
| 편집(Editorial) 디자인        | 책, 잡지, 신문 등 인쇄물의 레이아웃·타이포그래피·이미지 배열을 설계하는 디자인 |
| 광고 디자인                   | 제품이나 서비스를 알리기 위한 설득적 시각 커뮤니케이션 디자인                  |
| Typography                    | 문자의 서체, 크기, 간격, 배치 등을 통해 시각적 표현을 구성하는 분야            |
| 레터링(Lettering)             | 상업적 목적으로 문자를 손으로 직접 디자인하거나 변형하는 작업                  |
| Illustration                  | 메시지 전달을 위해 그림·회화적 이미지를 사용하는 시각 표현                     |
| CI(Corporate Identity)        | 기업 이미지를 통합적으로 관리하고 표현하는 시각 아이덴티티 시스템              |
| 심볼 / 로고 디자인            | 기업, 단체, 브랜드를 상징하는 간결한 시각적 마크 디자인                        |
| 포장(Package) 디자인          | 제품 보호와 판매 촉진을 위한 용기·포장재 디자인                                |
| POP(Point of Purchase) 디자인 | 판매 시점에서 구매를 유도하기 위한 매장 내 광고·진열 디자인                    |
| 영상 디자인                   | TV, 영화, 온라인 미디어 등 동영상 매체를 위한 시각 디자인                      |

**`포장 디자인의 요구 조건`**

| 조건       | 설명                                                     |
| ---------- | -------------------------------------------------------- |
| 보호보존성 | 내용물을 외부 충격, 오염, 변질로부터 보호하는 기능       |
| 편리성     | 운반, 개봉, 사용이 편리해야 하는 조건                    |
| 심미성     | 시각적으로 아름답고 매력적이어야 하는 조건               |
| 상품성     | 제품의 가치를 높이고 브랜드 이미지를 반영해야 하는 조건  |
| 구매의욕   | 소비자의 구매 욕구를 자극하고 판매를 촉진하는 조건       |
| 재활용성   | 사용 후 재활용 또는 친환경적으로 처리 가능해야 하는 조건 |

### 제품 디자인(Product Design)

제품 디자인은 제품의 목표와 범위, 기능과 디자인, 시장성, 사용자, 조형적 특성 및 조직 및 예산 등이 합리적이고 효과적으로 진행될 수 있도록 계획되어야 한다.

제품 디자인은 과학, 기술, 인간, 환경 등이 공존하는 분야이다.

생산 가능한 형태, 구조, 재료 등을 잘 선택한 설계여야 한다.

**`제품 디자인 의미`**

| 분류      | 설명                                                                 |
| --------- | -------------------------------------------------------------------- |
| 좁은 의미 | 공업 제품의 형태, 구조, 색채 등을 계획하는 인더스트리얼 디자인       |
| 넓은 의미 | 제품의 기획부터 생산, 판매까지 전 과정에 관여하는 포괄적 디자인 행위 |

**`제품 디자인 분류`**

| 분류                           | 설명                                                                    |
| ------------------------------ | ----------------------------------------------------------------------- |
| 생산 수단에 의한 분류          | 수공예 디자인(핸드크래프트)과 공업 디자인(인더스트리얼 디자인)으로 구분 |
| 도구적 장비의 성격에 의한 분류 | 가구, 가전제품, 수송 기기, 완구 등 사용 목적별로 구분                   |

### 환경 디자인(Environment Design)

환경 디자인이란 환경적인 문제인식을 가지며 사회와 자연의 관계를 통하여 쾌적한 환경 조성을 목적으로 하는 모든 디자인 분야를 포괄하는 상위 개념이다.

환경 디자인은 자연을 보호함으로써 인간을 보호하는 목적을 가진다.

**`환경 디자인 분야`**

| 분야                              | 설명                                                              |
| --------------------------------- | ----------------------------------------------------------------- |
| 도시 디자인                       | 도로, 광장, 공원 등 도시 공간 전체를 계획·설계하는 분야           |
| 실내(Interior) 디자인             | 건물 내부 공간의 기능성과 심미성을 높이기 위한 디자인 분야        |
| 디스플레이 디자인                 | 전시장, 쇼룸, 박람회 등 전시 공간을 연출하는 디자인 분야          |
| 스트리트 퍼니처(Street Furniture) | 가로등, 벤치, 버스 정류장 등 도시 시설물 디자인 분야              |
| 슈퍼 그래픽                       | 건물 외벽, 도로 등 대형 면적에 그래픽을 적용하는 환경 시각 디자인 |

## 디자인의 요소와 원리

### 디자인의 원리

디자인의 원리는 시각적으로 아름다움을 추구하고자 하는 것이다.

- 디자인의 목적은 인간의 행복이며, 바람직한 삶을 영위할 수 있게 하는 것이다.

디자인의 원리는 조화, 통일과 변화, 균형, 강조, 대비, 율동 등이 있다.

**`조화(Harmony)`**

조형 요소들이 서로 미묘하게 변화하여 균형을 잡고 있는 상태.

각 요소가 한 곳에 치우치지 않고 전체적으로 잘 어울리는 결합.

| 구분      | 설명                                        | 특징                   |
| --------- | ------------------------------------------- | ---------------------- |
| 유사 조화 | 비슷한 성질의 요소들을 결합하여 만드는 조화 | 부드럽고 안정적인 느낌 |
| 대비 조화 | 상반된 성질의 요소들을 결합하여 만드는 조화 | 강렬하고 역동적인 느낌 |

**`통일과 변화`**

통일은 각 요소들에 질서를 주어 조형적 일관성을 형성하는 것을 말한다.

변화는 통일과 상반된 개념으로, 요소들을 서로 다르게 구성하는 것이다.

| 구분          | 설명                                                               | 특징                       |
| ------------- | ------------------------------------------------------------------ | -------------------------- |
| 통일(Unity)   | 요소들에 질서와 일관성을 부여하여 전체가 하나로 느껴지게 하는 원리 | 안정적이고 통일된 느낌     |
| 변화(Variety) | 요소들에 다양성을 주어 시각적 흥미와 역동성을 만드는 원리          | 단조로움 방지, 생동감 부여 |

**`균형(Balance)`**

균형이란 시각적 무게의 동등한 힘의 분배를 주어, 디자인 요소들 간의 긴장감과 안정감을 유지하는 것이다.

균형은 대칭, 비대칭, 비례 등을 들 수 있다.

**대칭(Symmetry) 균형**

선이나 중앙을 기준으로 양쪽으로 같은 형태로 요소를 위치시키는 균형을 대칭 균형이라고 한다.

| 대칭 구분 | 설명                                                          |
| --------- | ------------------------------------------------------------- |
| 선 대칭   | 하나의 기준선(축)을 중심으로 양쪽이 거울처럼 대응하는 대칭    |
| 방사 대칭 | 중심점에서 여러 방향으로 동일한 패턴이 방사형으로 퍼지는 대칭 |
| 이동 대칭 | 동일한 형태가 일정 방향과 간격으로 반복 이동하는 대칭         |
| 확대 대칭 | 동일한 형태가 일정 비율로 크기를 달리하며 반복되는 대칭       |
| 역 대칭   | 형태를 180도 회전하거나 반전시켜 대응시키는 대칭              |

**비대칭(Asymmetry) 균형**

비대칭 균형은 개성적이며 감각적으로 느껴져, 대칭보다 비대칭적 구도의 활용도가 높다.

**비례(Proportion)**

요소들의 상대적 크기로, 수량적인 관계에 의한 균형을 말한다.

| 비례 구분     | 설명                                                                           |
| ------------- | ------------------------------------------------------------------------------ |
| 인체비례      | 인체의 비율을 기준으로 한 비례 체계 (예: 비트루비우스의 인체 비례)             |
| 황금비례      | 1:1.618의 황금비(Golden Ratio). 자연과 예술에서 가장 아름다운 비율로 여겨짐    |
| 모듈          | 기준 단위를 정해 반복 적용하는 비례 체계. 건축·디자인에서 규격화에 사용        |
| 피보나치 수열 | 앞 두 수의 합이 다음 수가 되는 수열(1, 1, 2, 3, 5, 8...). 황금비와 밀접한 관계 |
| 등차수열      | 일정한 간격(공차)으로 증가 또는 감소하는 수열에 기반한 비례                    |
| 루트비        | 정사각형의 대각선 비율(√2, √3 등)을 이용한 비례. A4 용지 등에 활용             |
| 금강비례      | 한국 전통 건축에서 사용된 1:√2 비례 체계                                       |

**`강조와 대비`**

강조란 요소들 중 한 가지 요소가 다른 성질로 인해 시각적 힘이 생겨 두드러지는 현상이다.

대비란 요소들 간의 조합으로 인해 상반되게 나타나는 현상이다.

**`율동`**

율동(Rhythm)이란 동일하거나 유사한 요소가 일정한 규칙과 질서를 가지며 배치되었을 때 나타나는 통일감이다.

| 율동 구분             | 설명                                                |
| --------------------- | --------------------------------------------------- |
| 반복                  | 동일한 요소나 패턴이 규칙적으로 되풀이되는 율동     |
| 교차                  | 두 가지 이상의 요소가 번갈아 반복되는 율동          |
| 방사                  | 중심점에서 바깥쪽으로 요소가 방사형으로 퍼지는 율동 |
| 점이(점증, Gradation) | 요소의 크기·색·형태 등이 점차적으로 변화하는 율동   |

## 디자인사

### 근대 디자인사

`미술공예운동(Arts and Crafts Movement)`

근대 디자인사에서 가장 먼저 일어난 운동.

18세기 산업혁명으로 인한 대량생산 방식으로 전환은 공예 간에 혼란을 야기시켰다. 이에 대항하여 미술과 공예의 회복 및 예술의 민주화를 주장한 운동.

| 구분     | 설명                                                                                                                                         |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지   | 영국                                                                                                                                         |
| 특징     | 기계 대량생산에 반대하고 수공예 전통 회복 주장<br>중세 고딕 양식과 자연(식물·유기적 형태)에서 영감<br>미술과 공예의 통합, 예술의 민주화 주장 |
| 주요인물 | 존 러스킨(John Ruskin) — 이론적 기반 제공<br>윌리엄 모리스(William Morris) — 실천적 리더                                                     |

`아르누보(Art Nouveau)`

'새로운 미술'이라는 뜻으로, 19세기 말에서 20세기 초에 벨기에와 프랑스를 중심으로 전 유럽으로 유행된 운동.

| 구분     | 설명                                                                                                                                                                                |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지   | 벨기에·프랑스(파리) 중심, 전 유럽으로 확산                                                                                                                                          |
| 특징     | 자연(식물, 꽃, 덩굴)에서 영감을 받은 유기적·곡선적 형태<br>장식성이 강하고 우아한 선(Line)의 미학<br>회화, 건축, 가구, 포스터, 공예 등 다양한 분야에 적용<br>실용성보다 장식성 중시 |
| 주요인물 | 알폰스 무하(Alphonse Mucha)<br>앙리 반 데 벨데(Henry van de Velde)<br>빅토르 오르타(Victor Horta)<br>에밀 갈레(Emile Gallé)                                                         |

`유겐트스틸(Jugendstil)`

'청춘양식'이라는 뜻으로, 독일식 아르누보로 독일 뮌헨에서 발행하는 미술 잡지 유겐트(Jugend)에서 유래되었다.

| 구분     | 설명                                                                                                                                                |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지   | 독일 뮌헨                                                                                                                                           |
| 특징     | 아르누보와 유사하게 자연적 형태와 장식성 강조<br>점차 기하학적 요소와 기능주의 방향으로 발전<br>독일공작연맹(DWB)·바우하우스로 이어지는 교량적 역할 |
| 주요인물 | 페터 베렌스(Peter Behrens)<br>헤르만 오브리스트(Hermann Obrist)<br>오토 엑만(Otto Eckmann)                                                          |

`분리파(Secession)`

과거의 전통양식으로부터 분리를 목적으로 1897년 오스트리아의 빈과 독일에서 일어난 예술운동.

| 구분     | 설명                                                                                                                                                                                               |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지   | 오스트리아 빈(Vienna Secession, 1897), 독일 뮌헨·베를린                                                                                                                                            |
| 특징     | 전통 아카데미즘으로부터 분리, 새로운 예술 추구<br>순수 예술과 응용 예술의 경계를 허물고 종합 예술 지향<br>기하학적 패턴·직선 강조 (아르누보의 곡선에서 탈피)<br>바우하우스 설립의 사상적 토대가 됨 |
| 주요인물 | 구스타프 클림트(Gustav Klimt)<br>요제프 호프만(Josef Hoffmann)<br>콜로만 모저(Koloman Moser)<br>요제프 마리아 올브리히(Joseph Maria Olbrich)                                                       |

`기능주의(Functionalism)`

디자인에 있어 아름다움보다 기능의 편리함과 만족함을 추구하는 미술사상.

19세기 후반 건축가들에 의해 도입되어 근대 건축의 발전에 큰 영향을 미쳤다.

**`기능주의 작가와 사상`**

| 작가                          | 주장                                                                                |
| ----------------------------- | ----------------------------------------------------------------------------------- |
| 오토 와그너                   | "예술은 오직 필요에만 기초를 두어야 한다" — 건축의 기능성과 실용성 강조             |
| 루이스 설리반(Louis Sullivan) | "형태는 기능을 따른다(Form follows function)" — 기능주의의 핵심 명제                |
| 르 꼬르뷔제                   | "집은 살기 위한 기계(A house is a machine for living in)" — 건축의 기계적 기능 강조 |
| 빅터 파파넥(Victor Papanek)   | "디자인은 사회적 책임을 가진다" — 인간 중심·친환경·소외 계층을 위한 디자인 강조     |
| 라이트(Frank Lloyd Wright)    | "유기적 건축(Organic Architecture)" — 자연환경과 조화를 이루는 건축 강조            |
| 그리노(Horatio Greenough)     | 자연에서 관찰한 기능적 형태의 아름다움 주창 — 기능이 형태를 결정한다                |

`독일공작연맹(DWB: Deutsher Werk Bund)`

독일에서 미술학교와 미술공예학교의 개혁이 성행되어 예술과 공예, 공업의 통일을 실현하는 것을 목표로 작업장 교육 구체화를 시도하려 했다. 이는 궁극적으로 교육기관과 산업을 결합시켜 나가려는 목적을 가졌다.

| 구분     | 설명                                                                                                                                                                                   |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지   | 독일 뮌헨 (1907년 설립)                                                                                                                                                                |
| 특징     | 예술·공예·공업의 통합으로 독일 제품 품질 향상 목표<br>예술가와 산업의 협력 추구<br>기계 생산을 적극 수용 (미술공예운동의 반기계 입장과 대조)<br>바우하우스(Bauhaus) 설립의 사상적 기반 |
| 주요인물 | 헤르만 무테지우스(Hermann Muthesius) — 설립 주도<br>페터 베렌스(Peter Behrens)<br>앙리 반 데 벨데(Henry van de Velde)                                                                  |

`입체주의(Cubism)`

1907 ~ 1914년에 걸쳐 프랑스 파리에서 일어난 미술 혁신 운동.

사실주의적 전통에서 벗어나 자연의 여러 가지 형태를 기본적인 기하학적 형태로 환원하고, 사물의 존재를 2차원적인 면의 분할로 재구성.

주요 인물: 파블로 피카소(Pablo Picasso), 조르주 브라크(Georges Braque), 페르낭 레제(Fernand Léger)

| 구분            | 특징                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------- |
| 초기 입체주의   | 폴 세잔의 영향을 받아 자연 형태를 기하학적 형태로 단순화 (1907~1909)                        |
| 분석적 입체주의 | 대상을 여러 시점에서 동시에 분석하여 면으로 해체·재구성. 단색조(갈색·회색) 사용 (1910~1912) |
| 종합적 입체주의 | 다양한 질감·색채와 콜라주(Collage) 기법 도입. 분석적 단계를 넘어 재구성 (1912~1914)         |

`구성주의(Constructivism)`

러시아 혁명기(1913 ~ 1920)에 일어난 운동.

과거 전통에서 벗어나서 현대적·기술적 원리의 예술을 주장.

디자인을 생산주의적 관점에서 이해하고, 저렴한 비용으로 최대한 효과를 얻을 수 있는 방향을 중요시함.

| 구분      | 설명                                                                                                                                                                |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지    | 러시아 (1913년~, 혁명 이후 소비에트 국가 예술로 발전)                                                                                                               |
| 특징      | 예술을 사회 혁명의 도구로 인식<br>기하학적 형태·직선·원·삼각형 등 추상적 구성 강조<br>산업 재료(철·유리·목재)를 적극 활용<br>기능주의와 연계하여 실용적 디자인 추구 |
| 주요 인물 | 알렉산드르 로드첸코(Alexander Rodchenko)<br>엘 리시츠키(El Lissitzky)<br>블라디미르 타틀린(Vladimir Tatlin)                                                         |

`데 스틸(De Stijl: 신조형주의)`

1917년 네덜란드에서 결성한 조형예술운동 그룹의 명칭으로, '양식(Style)'을 의미한다.

| 구분      | 설명                                                                                                                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 발생지    | 네덜란드 (1917년)                                                                                                                                                                          |
| 특징      | 수직·수평선과 삼원색(빨강·파랑·노랑), 무채색(흰·검·회)만 사용<br>순수 추상(신조형주의)을 지향하며 모든 장식 배제<br>그리드 구조에 기반한 기하학적 단순성<br>바우하우스에 직접적 영향을 줌  |
| 주요 인물 | 몬드리안(Piet Mondrian) — 회화에서 신조형주의 이론 정립<br>판 두스뷔르흐(Theo van Doesburg) — 그룹 창립 및 이론 주도<br>헤리트 리트펠트(Gerrit Rietveld) — 레드블루 의자 등 가구·건축 적용 |

`바우하우스(Bauhaus)`

독일의 바이마르에 창립한 종합 조형학교.

| 구분      | 설명                                                                                                                                                                                                                                |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 발생지    | 독일 바이마르(1919년 창립) → 데사우 → 베를린으로 이전                                                                                                                                                                               |
| 특징      | 예술·공예·기술의 통합을 지향한 종합 조형학교<br>워크숍 중심의 실습 교육 방식<br>기능주의·합리주의 디자인 철학<br>나치 탄압으로 1933년 강제 폐교                                                                                     |
| 목표      | 순수 예술과 응용 예술의 통합<br>대량 생산에 적합한 기능적 디자인 실현                                                                                                                                                               |
| 주요 인물 | 발터 그로피우스(Walter Gropius) — 창립자<br>파울 클레(Paul Klee), 바실리 칸딘스키(Wassily Kandinsky)<br>마르셀 브로이어(Marcel Breuer), 라슬로 모호이나기(László Moholy-Nagy)<br>미스 반 데어 로에(Mies van der Rohe) — 마지막 교장 |

| 시대별 특성 | 년도      | 명칭             | 특징                                                           |
| ----------- | --------- | ---------------- | -------------------------------------------------------------- |
| 1기         | 1919~1923 | 바이마르(Weimar) | 발터 그로피우스 지도. 예술과 공예의 통합 실험, 표현주의적 성격 |
| 2기         | 1925~1928 | 데사우(Dessau)   | 발터 그로피우스 지도. 기능주의·합리주의 확립, 산업 생산과 결합 |

`아르데코(Art Deco)`

1925년 파리 장식 미술박람회에서 나온 명칭으로, '장식미술'을 의미하는 'Art Decoratif'의 약칭이다.

기능적이고 고전적인 직선미를 추구하여, 직선, 동심원, 기하학적인 형태와 반복 패턴을 선호하여 기계제품에 적합하도록 대중화를 중시한 조형운동이다.

| 구분      | 설명                                                                                                                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 발생지    | 프랑스 파리 (1925년 장식 미술박람회에서 명칭 유래)                                                                                                                                         |
| 특징      | 기하학적 직선, 동심원, 지그재그, 반복 패턴 선호<br>아르누보의 유기적 곡선에서 탈피한 직선미 추구<br>고급 소재(금속, 상아, 에나멜)와 화려한 색채<br>기계 미학 수용, 대량 생산에 적합한 장식 |
| 목표      | 기능성과 장식성의 조화<br>현대적·산업적 라이프스타일 반영                                                                                                                                  |

## 디자인과 마케팅

### 디자인 정책 및 관리

**`디자인 정책(Design policy)`**

제품을 어떠한 방법으로 소비자에게 알리고 판매할 것인지 결정하는 통합 정책.

이후 기업 이미지 통합 프로그램(CIP, Corporate Identity Program)으로 발전하였다.

**`디자인 관리(Design management)`**

디자인 정책을 시행하고 관리하는 전반적인 프로세스를 디자인 관리라고 한다.

기업의 목표와 경영진의 의사를 반영한 합리적인 디자인을 도출하기 위해 경영진과 디자이너 사이에 원활한 소통이 이루어져야 한다.

**`디자인 관리에 대한 디자이너 의식`**

미적 표현 기능만을 고집하지 말아야 한다.

개성적인 이미지보다 객관적인 이미지를 표현해야 한다.

경영학, 마케팅, 심리학, 광고학, 인간공학 등 폭넓은 지식을 갖추어야 한다.

디자인 분야에 대한 직무가 아닌, 경영 분야도 포함한 마케터적 역할을 수행해야 한다.

### 마케팅의 정의, 기능

마케팅이란, 소비자의 욕구를 철저히 조사하여 소비자가 원하는 상품 및 서비스를 개발하고 합리적인 유통경로와 가격을 통하여 소비자에게 제공하며 판매촉진을 일으켜 소비자의 만족과 기업의 이윤 추구를 위한 기업의 총체적 활동이다.

| 구분 | 설명                                                                    |
| ---- | ----------------------------------------------------------------------- |
| 역할 | 소비자 욕구 조사 &rarr; 서비스 개발 &rarr; 합리적 유통경로 및 가격 결정 |
| 목적 | 소비자의 만족 및 기업의 이윤 추구                                       |

**`마케팅 구분`**

| 종류        | 설명                                                                            |
| ----------- | ------------------------------------------------------------------------------- |
| 거시 마케팅 | 사회 전체의 관점에서 마케팅 시스템이 사회에 미치는 영향을 분석하는 마케팅       |
| 미시 마케팅 | 개별 기업이 특정 목표 시장을 대상으로 수행하는 마케팅 활동                      |
| 니치 마케팅 | 틈새시장(Niche Market)을 공략하여 소수 집단의 욕구를 집중적으로 충족하는 마케팅 |

| 원칙                   | 설명                                                                         |
| ---------------------- | ---------------------------------------------------------------------------- |
| 수요전제의 원칙        | 마케팅은 소비자의 수요를 전제로 이루어진다는 원칙                            |
| 판매촉진의 원칙        | 판매 촉진을 마케팅의 핵심 목표로 삼는다는 원칙                               |
| 유통계열화의 원칙      | 생산자에서 소비자까지 유통 경로를 체계적으로 계열화하여 효율을 높인다는 원칙 |
| 기업주체성의 원칙      | 마케팅 활동의 주체는 기업이며, 기업이 주도적으로 시장을 창출한다는 원칙      |
| 판매중추성의 원칙      | 판매가 마케팅 활동의 중심이 되어야 한다는 원칙                               |
| 과학적 시장인식의 원칙 | 시장을 주관적 감각이 아닌 데이터와 과학적 방법으로 분석해야 한다는 원칙      |

### 마케팅 믹스(Marketing Mix)

**`4P 전략 - 기업 관점`**

| 요소            | 설명                                                                                       |
| --------------- | ------------------------------------------------------------------------------------------ |
| 제품(Product)   | 소비자 욕구를 충족시키기 위해 기업이 제공하는 상품·서비스. 품질·디자인·브랜드·포장 등 포함 |
| 가격(Price)     | 제품·서비스에 부여하는 금전적 가치. 원가·경쟁사 가격·소비자 인식 등을 고려하여 결정        |
| 유통(Place)     | 제품이 생산자에서 최종 소비자에게 전달되는 경로 및 채널. 물류·유통망 관리 포함             |
| 촉진(Promotion) | 소비자에게 제품을 알리고 구매를 유도하는 활동. 광고·홍보·판촉·인적 판매 등 포함            |

**`4C 전략 - 소비자 관점`**

| 요소                             | 설명                                                                                                                  |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| 소비자 해결책(Customer Solution) | 기업이 만든 제품(Product)이 소비자에게 어떤 가치를 주고 문제를 해결해 주는가                                          |
| 소비자 비용(Customer Cost)       | 제품의 가격(Price)을 넘어, 소비자가 이를 구매하고 이용하기 위해 지불하는 시간과 노력 등을 포함한 전체 비용은 얼마인가 |
| 편리성(Convenience)              | 매장 방문(Place)의 불편함 없이, 소비자가 얼마나 쉽고 편리하게 이 제품을 구매할 수 있는가                              |
| 소통(Communication)              | 일방적인 광고(Promotion)가 아닌, 소셜 미디어를 통해 소비자와 얼마나 쌍방향으로 잘 소통하는가                          |

### 제품 수명 주기 분석(Product Life Cycle)

제품 아이디어 창출부터 사라질 때 까지의 판매의 규모와 이익 등을 고려하여 분석하는 것을 제품 수명 주기 분석이라고 한다.

| 단계           | 설명                                                                                                            |
| -------------- | --------------------------------------------------------------------------------------------------------------- |
| 도입기         | 제품이 처음 시장에 출시되는 단계. 매출이 낮고 초기 투자 비용이 높아 이익 실현이 어려움. 인지도 구축이 핵심 과제 |
| 성장기         | 소비자 인지도가 높아지고 매출이 급격히 증가하는 단계. 경쟁자가 진입하기 시작하며 이익이 빠르게 증가             |
| 경쟁기(성숙기) | 매출 성장이 둔화되고 시장이 포화 상태에 이르는 단계. 경쟁이 가장 치열하며 이익이 정점에서 서서히 감소           |
| 쇠퇴기         | 매출과 이익이 지속적으로 감소하는 단계. 소비자 관심 저하 또는 대체품·신제품 등장으로 시장에서 퇴출              |

**`시장의 유행성`**

시장의 유행은 매출과 유행 기간에 따라 4단계로 구분된다.

| 구분    | 설명                                                                                         |
| ------- | -------------------------------------------------------------------------------------------- |
| Flop    | 출시 즉시 실패하고 사라지는 제품. 매출이 거의 발생하지 않음                                  |
| Fad     | 갑자기 폭발적인 인기를 얻었다가 빠르게 사라지는 단기 유행                                    |
| Ford    | 대중에게 안정적으로 수용되어 꾸준한 매출이 지속되는 중·장기 유행 (Fashion으로 표기하기도 함) |
| Classic | 시간이 지나도 꾸준히 사랑받는 스테디셀러형 장기 스타일                                       |

### 마케팅 전략

**`SWOT 분석`**

기업의 내부 여건(Strength / Weakness)과 외부 여건(Opportunity / Threat)을 분석하여 전략적인 대안을 선택할 수 있다.

| 구분               | 설명                                                         |
| ------------------ | ------------------------------------------------------------ |
| 강점-기회(SO) 전략 | 내부 강점을 활용하여 외부 기회를 최대한 포착하는 공격적 전략 |
| 강점-위협(ST) 전략 | 내부 강점으로 외부 위협을 극복하거나 회피하는 다양화 전략    |
| 약점-기회(WO) 전략 | 내부 약점을 보완하여 외부 기회를 활용하는 방향 전환 전략     |
| 약점-위협(WT) 전략 | 내부 약점을 최소화하고 외부 위협을 회피하는 방어적·축소 전략 |

**`포지셔닝(Positioning)`**

선정된 **표적 시장(Target Market)** 내에서 자사 제품이나 브랜드를 소비자의 마음속(Mind)에 차별화된 고유한 위치로 인식시키는 전략적 과정을 포지셔닝이라고 한다.

제품의 물리적 위치가 아닌, 경쟁사 대비 '소비자의 지각 공간' 내에서의 상대적 위치를 선점한다.

- 자사 및 경쟁사 표적 시장 위치 파악 &rarr; 전략적 위치 결정

**`포지셔닝 맵(Positioning Map)`**

자사와 타사의 표적 시장 내 위치를 파악하기 위해, 소비자의 구매 결정 기준(속성)을 축으로 삼아 시각적으로 표현한 도표를 **포지셔닝 맵(Positioning Map)** 이라고 한다.

- 학술적으로는 **지각도(Perceptual Map)**라고도 부른다.

**`마케팅 전략`**

| 전략                            | 설명                                                                                             |
| ------------------------------- | ------------------------------------------------------------------------------------------------ |
| 시장 침투 전략                  | 기존 제품으로 기존 시장에서 점유율을 높이는 전략. 가격 인하·판촉 강화 등 활용                    |
| 제품 개발 전략                  | 기존 시장에 새로운 제품을 출시하여 성장하는 전략                                                 |
| 시장 개발 전략                  | 기존 제품을 새로운 시장(신규 고객·지역)에 진출시키는 전략                                        |
| 다각화 전략                     | 새로운 제품으로 새로운 시장에 진입하는 전략. 위험은 높지만 성장 가능성이 큼                      |
| 차별화 전략(Differentiation)    | 경쟁사와 차별되는 독특한 가치·품질·브랜드를 제공하여 우위를 점하는 전략                          |
| 원가 우위 전략(Cost Leadership) | 낮은 생산 비용으로 경쟁사보다 저렴한 가격을 제공하는 전략                                        |
| 집중화 전략(Focus)              | 특정 세분 시장에 자원을 집중하여 경쟁력을 확보하는 전략                                          |
| 블루오션 전략                   | 경쟁이 치열한 기존 시장(레드오션)을 벗어나 경쟁자 없는 새로운 시장 공간을 창출하는 전략          |
| 2등(Fast Second) 전략           | 시장 개척자(First Mover)의 시행착오를 교훈 삼아 빠르게 2위로 진입하여 선두를 역전하는 전략       |
| 퍼플카우(Purple Cow) 전략       | 세스 고딘(Seth Godin)이 제안. 제품·서비스 자체를 보라색 소처럼 눈에 띄고 화제가 되게 만드는 전략 |
| MOT(Moment Of Truth) 마케팅     | 고객이 기업·제품과 접촉하는 순간(진실의 순간)에 긍정적 경험을 제공하여 충성도를 높이는 마케팅    |
| DM(Direct Mail) 광고 마케팅     | 우편·이메일 등을 통해 특정 고객에게 직접 광고물을 발송하는 직접 마케팅 방식                      |
| 공동상표 마케팅                 | 두 개 이상의 브랜드가 협력하여 공동 마케팅 또는 공동 브랜드를 활용하는 전략 (Co-branding)        |

### 시장조사

시장조사는 상품 및 서비스의 마케팅과 관련한 문제에 관한 자료를 조직적으로 수집, 기록, 분석하는 것을 의미한다.

- 시장조사는 마케팅 조사라고도 한다.

**`조사접근 방법`**

| 종류                                      | 설명                                                              |
| ----------------------------------------- | ----------------------------------------------------------------- |
| 관찰법                                    | 조사자가 직접 현장에서 소비자의 행동·반응을 관찰·기록하는 방법    |
| 실험법                                    | 변수를 통제한 환경에서 자극과 반응의 인과관계를 측정하는 방법     |
| 설문조사                                  | 구조화된 질문지를 통해 다수의 응답자로부터 데이터를 수집하는 방법 |
| 표적 집단조사(FGI, Focus Group Interview) | 소수의 대상자를 모아 특정 주제에 대해 심층 토론하게 하는 방법     |

**`실사조사 방법`**

| 종류        | 설명                                                                                        |
| ----------- | ------------------------------------------------------------------------------------------- |
| 개인 면접법 | 조사원이 응답자와 직접 대면하여 질문·응답하는 방법. 심층 정보 수집에 유리하나 비용이 높음   |
| 전화 면접법 | 전화를 통해 응답자에게 질문하는 방법. 신속하고 저렴하나 응답 거부율이 높을 수 있음          |
| 우편 조사법 | 설문지를 우편으로 발송하여 응답을 수거하는 방법. 광범위한 대상 가능하나 회수율이 낮음       |
| 관찰 조사법 | 응답자의 행동·환경을 직접 또는 기기를 통해 관찰하는 방법. 자연스러운 행동 데이터 수집 가능  |
| 그룹 인터뷰 | 소수의 참가자가 모여 특정 주제에 대해 자유롭게 토론하는 방법. 심층 의견 및 태도 파악에 유리 |

### 소비자 생활유형(Life style)

**`소비자 행동에 미치는 요인`**

| 요인        | 설명                                                                               |
| ----------- | ---------------------------------------------------------------------------------- |
| 문화적 요인 | 문화, 하위문화, 사회 계층 등 사회가 공유하는 가치관·관습이 구매 행동에 미치는 영향 |
| 사회적 요인 | 준거집단(행동 기준으로 삼는 집단)<br>대면집단(실제 접촉하는 집단)                  |
| 개인적 요인 | 나이, 직업, 소득 수준, 라이프스타일, 개성 등 개인의 특성이 구매 행동에 미치는 영향 |
| 심리적 요인 | 동기, 지각, 학습, 신념·태도 등 내면적 심리 상태가 구매 행동에 미치는 영향          |

**`소비자 유형과 특성`**

| 집단             | 설명                                                                    |
| ---------------- | ----------------------------------------------------------------------- |
| 관습적 집단      | 익숙한 브랜드·제품을 반복 구매하며 변화를 거부하는 보수적 소비자        |
| 감성적 집단      | 이미지, 디자인, 감각적 요소를 중시하며 감정·직관에 따라 구매하는 소비자 |
| 합리적 집단      | 품질, 가격, 성능을 꼼꼼히 비교·분석하여 이성적으로 구매하는 소비자      |
| 유동적 집단      | 유행과 트렌드에 민감하게 반응하며 구매 패턴이 자주 변하는 소비자        |
| 가격 중심적 집단 | 가격을 최우선 기준으로 삼아 최저가·할인 상품을 선호하는 소비자          |
| 신 소비 집단     | 새로운 제품·서비스·경험을 추구하며 혁신을 적극적으로 수용하는 소비자    |

**`소비자 구매 심리 과정`**

AIDMA 모델 — 소비자가 광고·제품을 접한 후 실제 구매에 이르기까지의 심리 단계.

| 과정            | 설명                                                        |
| --------------- | ----------------------------------------------------------- |
| 주의(Attention) | 광고·제품을 통해 소비자의 시선과 주의를 끄는 단계           |
| 흥미(Interest)  | 제품에 대한 관심과 흥미가 발생하는 단계                     |
| 욕구(Desire)    | 제품을 소유하거나 사용하고 싶다는 구체적 욕구가 생기는 단계 |
| 기억(Memory)    | 제품 정보나 브랜드를 기억하여 인지도가 형성되는 단계        |
| 행동(Action)    | 실제 구매 행동으로 이어지는 단계                            |

### 소비자 생활유형 측정

**`AIO법`**

사람들이 시간을 보내는 방식, 관심, 의견을 파악하기 위한 측정법.

| 요소           | 설명                                                                |
| -------------- | ------------------------------------------------------------------- |
| 활동(Activity) | 여가, 쇼핑, 스포츠, 직업 등 소비자가 실제로 시간을 보내는 행동 방식 |
| 흥미(Interest) | 가족, 패션, 음식, 미디어 등 소비자가 특별한 관심을 기울이는 대상    |
| 의견(Opinion)  | 사회, 정치, 제품 등에 대한 소비자의 견해와 생각                     |

**`VALS(Values And Life Style)법`**

시장의 세분화를 위해 생활유형을 측정하는 방법. 미국 SRI(스탠포드 연구소)가 개발한 분류 체계.

| 유형          | 설명                                                                                                               |
| ------------- | ------------------------------------------------------------------------------------------------------------------ |
| 욕구지향 유형 | 생존에 필요한 기본 욕구 충족에 초점. 경제적으로 제한된 환경에서 가격 중심적 구매 (생존자형·지속자형)               |
| 외부지향 유형 | 타인의 시선과 사회적 인정을 중시. 소속감·지위·성취를 추구하며 사회적 규범에 따라 행동 (귀속자형·경쟁자형·성취자형) |
| 내부지향 유형 | 자신의 가치관과 내면적 기준으로 행동. 자아실현과 개인적 성장을 중시 (개인주의형·경험추구형·통합형)                 |

### 친환경 디자인과 마케팅

| 구분                    | 특징                                                                                                                                            |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| 에코 디자인(Eco Design) | 제품 기획·설계 단계부터 환경 영향을 최소화하는 디자인 방법론. 원재료 절약, 에너지 효율, 재활용성, 유해물질 배제 등 제품 전 생애주기(LCA)를 고려 |
| 친환경 마케팅           | 환경 보호를 기업의 사회적 가치와 연결하여 소비자의 환경 의식을 마케팅 전략에 활용하는 포괄적 접근법                                             |
| 그린 마케팅             | 환경 친화적 제품·서비스를 개발하고 환경 보호 메시지로 홍보하는 마케팅 전략. 허위·과장 홍보인 그린워싱(Greenwashing)과 구별 필요                 |

## 시각 디자인

### 광고 디자인

**`광고의 구성 요소`**

| 요소                         | 설명                                                                              |
| ---------------------------- | --------------------------------------------------------------------------------- |
| 헤드라인(Head Line)          | 광고에서 가장 눈에 띄는 핵심 문구. 독자의 시선을 끌고 광고 내용을 압축적으로 전달 |
| 서브 헤드라인(Sub-Head Line) | 헤드라인을 보완하는 보조 문구. 추가 정보나 맥락을 제공                            |
| 바디 카피(Body Copy)         | 광고의 본문 텍스트. 제품·서비스의 특징과 장점을 상세히 설명                       |
| 캡션(Caption)                | 사진, 그림, 도표 등 이미지에 붙이는 설명 문구                                     |
| 캐치프레이즈(Catch-Pharse)   | 소비자의 관심을 끌기 위한 짧고 인상적인 문구. 특정 광고 캠페인에 한정 사용        |
| 슬로건(Slogan)               | 기업·브랜드의 이념이나 가치관을 담은 일관된 문구. 장기적으로 반복 사용            |

**`Catch-Pharse / Slogan 차이점`**

| 구분   | Catch-Phrase                                 | Slogan                                    |
| ------ | -------------------------------------------- | ----------------------------------------- |
| 목적   | 특정 광고 캠페인에서 소비자의 주목을 끄는 것 | 기업·브랜드의 이념·가치를 장기적으로 각인 |
| 지속성 | 단기 캠페인 한정                             | 장기적·반복적으로 사용                    |
| 범위   | 개별 광고 단위                               | 브랜드·기업 전체 수준                     |

**`조형적 요소`**

| 요소         | 설명                                                        |
| ------------ | ----------------------------------------------------------- |
| Illustration | 광고 메시지를 시각적으로 전달하기 위한 그림·회화적 이미지   |
| Symbol mark  | 기업·브랜드를 상징하는 추상적·구상적 도형 마크              |
| Trademark    | 상품이나 서비스를 식별하기 위해 법적으로 등록된 상표 마크   |
| Logo Type    | 기업·브랜드명을 특정 서체와 디자인으로 표현한 문자형 로고   |
| Border Line  | 광고 영역의 경계를 구분하거나 레이아웃을 정리하는 테두리 선 |

### 신문 광고

**`신문 광고 종류`**

| 종류                | 설명                                                                |
| ------------------- | ------------------------------------------------------------------- |
| 보도란 광고         | 신문의 기사 지면(보도란) 사이에 게재되는 일반적인 형태의 광고       |
| 광고란 광고         | 광고 전용 지면에 게재되는 광고. 독자가 광고임을 인식하고 보는 지면  |
| 변형 광고           | 일반적인 직사각형 형태를 벗어나 특수한 모양으로 제작된 광고         |
| 티저(Teaser)        | 제품·브랜드를 직접 공개하지 않고 궁금증을 유발하여 관심을 끄는 광고 |
| 멀티 광고           | 동일 지면에 여러 광고를 연속·병렬로 배치한 광고 형식                |
| 타이업(Tie-Up) 광고 | 두 개 이상의 기업·브랜드가 협력하여 공동으로 진행하는 광고          |

### 잡지 광고

**`잡지 광고 종류`**

| 종류              | 설명                                                                         |
| ----------------- | ---------------------------------------------------------------------------- |
| 표지 1면 광고     | 잡지의 앞표지 전면에 게재되는 광고. 노출도가 가장 높아 광고 단가가 가장 높음 |
| 표지 2면 광고     | 표지 안쪽 첫 번째 페이지(표지를 펼쳤을 때 왼쪽 면)에 게재되는 광고           |
| 표지 3면 광고     | 뒷표지 안쪽 면(맨 뒤에서 두 번째 페이지)에 게재되는 광고                     |
| 표지 4면 광고     | 잡지의 뒷표지 전면에 게재되는 광고. 앞표지 다음으로 노출도가 높음            |
| 차례(목차면) 광고 | 목차 페이지에 게재되는 광고. 독자 열독률이 높은 위치                         |
| 기사 중 광고      | 본문 기사 사이에 삽입되는 광고. 기사와 함께 자연스럽게 읽힘                  |
| 표지 대면 광고    | 앞표지와 마주 보는 첫 번째 본문 페이지에 게재되는 광고                       |
| 안내 광고         | 분류·안내 형식으로 작은 면적에 여러 광고주가 나열된 광고                     |

**`열독률(Reader Engagement)`**

열독률이란 광고를 실제로 읽은 독자의 비율. 인쇄 광고 효과를 측정하는 지표.

| 구분   | 설명                                         |
| ------ | -------------------------------------------- |
| 인지율 | 광고를 본 적 있다고 응답한 독자의 비율       |
| 정독률 | 광고를 어느 정도 읽었다고 응답한 독자의 비율 |
| 완독률 | 광고 전체를 끝까지 읽은 독자의 비율          |

### TV 광고

**`TV 광고 종류`**

| 종류              | 설명                                                                        |
| ----------------- | --------------------------------------------------------------------------- |
| Spot 광고         | 프로그램과 프로그램 사이 시간대에 방영되는 광고. 가장 일반적인 TV 광고 형태 |
| Program 광고      | 특정 프로그램의 방영 전후에 게재되는 광고. 해당 프로그램 시청자를 직접 타겟 |
| Sponsor Ship 광고 | 기업이 프로그램 제작비를 후원하고 프로그램 내·전후에 광고하는 형식          |
| Local 광고        | 전국 방송이 아닌 특정 지역 방송국에서만 방영되는 지역 한정 광고             |
| 자막 광고         | 방송 화면 하단이나 측면에 텍스트 형태로 표시되는 광고                       |

### POP(Point of Purchase : 구매시점) 광고

소비자가 실제로 구매 결정을 내리는 판매 현장에서 제공되는 광고·판촉물.

| 종류              | 설명                                             |
| ----------------- | ------------------------------------------------ |
| 포스터            | 매장 내 벽·창문에 부착하는 광고물                |
| 쇼카드(Showcard)  | 상품 정보·가격·특징을 알리는 카드형 광고물       |
| 모빌(Moblie)      | 천장에 매달아 움직임으로 시선을 끄는 입체 광고물 |
| 스티커            | 상품·진열대에 부착하는 소형 광고물               |
| 디스플레이 스탠드 | 매장 내 독립형 진열·광고 구조물                  |

### DM(Direct mail) 광고

**`DM 광고 종류`**

| 종류                      | 설명                                                                        |
| ------------------------- | --------------------------------------------------------------------------- |
| 엽서                      | 엽서 형태의 소형 DM. 간단한 메시지·이미지를 담아 발송하는 가장 단순한 형식  |
| 폴더(Folder)              | 한 장의 종이를 한 번 이상 접어 만든 인쇄물. 간단한 정보 정리에 적합         |
| 소책자(Booklet)           | 여러 페이지로 구성된 작은 책자 형태. 상세한 제품·서비스 정보 전달에 적합    |
| 리플릿(Leaflet)           | 한 장의 종이를 여러 번 접어 만든 소형 인쇄물. 간단한 안내·광고에 활용       |
| 세일즈 레터(Sales Letter) | 개인에게 발송되는 편지 형식의 DM. 구매를 유도하는 설득적 문구 사용          |
| Broadside                 | 한 장의 큰 종이 양면에 인쇄한 대형 DM. 접어서 발송하며 펼치면 포스터 형태   |
| 노벨티(Novelty)           | 실용적·기념품적 가치가 있는 소형 홍보물 (달력, 볼펜 등). 브랜드를 지속 노출 |
| 블로터(Blotter)           | 잉크 흡수지에 인쇄한 홍보물. 실용적 용도(잉크 흡수)와 광고 기능을 겸함      |

### 편집 디자인(Editorial Design)

인쇄물을 시각적으로 구성하여 제작하는 시각 디자인의 한 분야.

출판 디자인(Publication Design) 또는 Editorial Design으로도 불린다.

**`형태별 분류`**

| 분류                  | 설명                                                                         |
| --------------------- | ---------------------------------------------------------------------------- |
| 낱장(Sheet) 형식      | 한 장의 종이 위에 정보를 배치하는 형식 (포스터, 전단지 등)                   |
| 스프레드(Spread) 형식 | 두 페이지를 펼쳐 하나의 화면으로 구성하는 형식 (잡지 양면 레이아웃 등)       |
| 서적 형식             | 여러 페이지를 제본하여 책의 형태로 구성하는 형식 (단행본, 잡지, 카탈로그 등) |

**`출판물 종류`**

| 종류               | 설명                                                                                    |
| ------------------ | --------------------------------------------------------------------------------------- |
| 학술지             | 학문적 연구 결과를 게재하는 정기 간행물                                                 |
| 문학지             | 소설, 시, 수필 등 문학 작품을 수록하는 정기 간행물                                      |
| 단행본             | 독립된 주제를 다루는 단권 도서                                                          |
| 매뉴얼(Manual)     | 제품 사용법이나 업무 절차를 설명하는 지침서                                             |
| 브로슈어(Brochure) | 기업·제품·서비스를 소개하는 고품질 인쇄 안내물. 리플릿보다 페이지가 많고 고급 소재 사용 |
| 리플릿(Leaflet)    | 간단한 정보를 담은 소형 접지 인쇄물                                                     |
| 카탈로그(Catalog)  | 제품 목록과 상세 정보를 체계적으로 정리한 인쇄물                                        |

**`인쇄 기법`**

| 분류        | 방법                                                | 종류                    | 장점                                       | 단점                           |
| ----------- | --------------------------------------------------- | ----------------------- | ------------------------------------------ | ------------------------------ |
| 볼록판 인쇄 | 잉크가 묻는 볼록한 부분을 종이에 눌러 찍는 방식     | 활판 인쇄, 플렉소그래피 | 내구성이 좋고 선명한 인쇄 가능             | 정교한 이미지·계조 표현에 한계 |
| 오목판 인쇄 | 오목하게 파인 홈에 잉크를 채워 종이에 옮기는 방식   | 그라비아 인쇄, 에칭     | 색조·계조 표현이 뛰어나고 대량 인쇄에 유리 | 제판 비용이 가장 높음          |
| 평판 인쇄   | 기름과 물의 반발 원리를 이용하며 판면이 평평한 방식 | 오프셋 인쇄, 리소그래피 | 대량 인쇄에 적합하고 품질 우수             | 초기 제판 비용이 높음          |
| 공판 인쇄   | 판의 구멍(망)을 통해 잉크를 밀어내는 방식           | 실크스크린, 스텐실      | 다양한 소재에 인쇄 가능, 후가공 효과 활용  | 대량 인쇄 속도가 느림          |

### 포장 디자인(Package Design)

상품의 신분을 확립하고, 소비자의 충동구매를 유발하며 안전하게 사용할 수 있도록 하는 분야를 포장 디자인이라고 한다.

- 패키지 디자인이라고도 한다.

**`포장 디자인 매체`**

| 매체            | 설명                                                                                            |
| --------------- | ----------------------------------------------------------------------------------------------- |
| POP 포장 디자인 | 판매 시점(매장 진열대)에서 소비자의 구매를 유도하기 위해 디자인된 포장물. 구매 결정에 직접 영향 |
| 포장지 디자인   | 선물이나 상품을 감싸는 포장지 자체를 디자인하는 분야. 브랜드 이미지 전달과 심미적 가치 제공     |
| 쇼핑백 디자인   | 소비자가 구매 상품을 담아 가는 쇼핑백을 디자인하는 분야. 브랜드 홍보 효과와 재사용성 고려       |
| 라벨 디자인     | 제품 용기·포장재에 부착되는 라벨을 디자인하는 분야. 브랜드 정보, 성분, 사용법 등을 포함         |
