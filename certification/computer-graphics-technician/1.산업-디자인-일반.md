## 디자인 일반

### 디자인의 4단계 성립과정

| 과정     | 설명                                                      |
| -------- | --------------------------------------------------------- |
| 욕구과정 | 인간의 필요와 욕구를 파악하고 디자인 목표를 설정하는 과정 |
| 조형과정 | 아이디어를 시각화·형태화하는 조형적 표현 과정             |
| 재료과정 | 목적에 맞는 재료를 선택하고 검토하는 과정                 |
| 기술과정 | 선정된 재료를 가공·제작하기 위한 기술을 적용하는 과정     |

### 산업 디자인 영역

산업 디자인은 크게 시각 디자인, 제품 디자인, 환경 디자인으로 구분된다

### 시각 디자인(Visual Design)

시각 전달(Visual Communication)디자인, 커뮤니케이션 디자인 등으로 쓰이고 있다.

사람과 사람 사이의 기호에 의해 의미를 전달하는 과정이다.

| 기능          | 시각 디자인 종류                 |
| ------------- | -------------------------------- |
| 지시적 기능   | 교통 표지판, 안내 표지, 픽토그램 |
| 설득적 기능   | 광고, 포스터, 홍보물             |
| 상징적 기능   | 심볼 마크, 로고, 캐릭터          |
| 기록적인 기능 | 사진, 다큐멘터리, 영상 기록물    |

**`시각 디자인 분류`**

| 분류                          | 설명                                                                           |
| ----------------------------- | ------------------------------------------------------------------------------ |
| 그래픽 디자인                 | 이미지, 문자, 색상 등을 조합해 시각적 커뮤니케이션을 구성하는 디자인           |
| 편집(Editorial) 디자인        | 책, 잡지, 신문 등 인쇄물의 레이아웃·타이포그래피·이미지 배열을 설계하는 디자인 |
| 광고 디자인                   | 제품이나 서비스를 알리기 위한 설득적 시각 커뮤니케이션 디자인                  |
| Typography                    | 문자의 서체, 크기, 간격, 배치 등을 통해 시각적 표현을 구성하는 분야            |
| 레터링(Lettering)             | 상업적 목적으로 문자를 손으로 직접 디자인하거나 변형하는 작업                  |
| Illustration                  | 메시지 전달을 위해 그림·회화적 이미지를 사용하는 시각 표현                     |
| CI(Corporate Identity)        | 기업 이미지를 통합적으로 관리하고 표현하는 시각 아이덴티티 시스템              |
| 심볼 / 로고 디자인            | 기업, 단체, 브랜드를 상징하는 간결한 시각적 마크 디자인                        |
| 포장(Package) 디자인          | 제품 보호와 판매 촉진을 위한 용기·포장재 디자인                                |
| POP(Point of Purchase) 디자인 | 판매 시점에서 구매를 유도하기 위한 매장 내 광고·진열 디자인                    |
| 영상 디자인                   | TV, 영화, 온라인 미디어 등 동영상 매체를 위한 시각 디자인                      |

**`포장 디자인의 요구 조건`**

| 조건       | 설명                                                     |
| ---------- | -------------------------------------------------------- |
| 보호보존성 | 내용물을 외부 충격, 오염, 변질로부터 보호하는 기능       |
| 편리성     | 운반, 개봉, 사용이 편리해야 하는 조건                    |
| 심미성     | 시각적으로 아름답고 매력적이어야 하는 조건               |
| 상품성     | 제품의 가치를 높이고 브랜드 이미지를 반영해야 하는 조건  |
| 구매의욕   | 소비자의 구매 욕구를 자극하고 판매를 촉진하는 조건       |
| 재활용성   | 사용 후 재활용 또는 친환경적으로 처리 가능해야 하는 조건 |

### 제품 디자인(Product Design)

제품 디자인은 제품의 목표와 범위, 기능과 디자인, 시장성, 사용자, 조형적 특성 및 조직 및 예산 등이 합리적이고 효과적으로 진행될 수 있도록 계획되어야 한다.

제품 디자인은 과학, 기술, 인간, 환경 등이 공존하는 분야이다.

생산 가능한 형태, 구조, 재료 등을 잘 선택한 설계여야 한다.

**`제품 디자인 의미`**

| 분류      | 설명                                                                 |
| --------- | -------------------------------------------------------------------- |
| 좁은 의미 | 공업 제품의 형태, 구조, 색채 등을 계획하는 인더스트리얼 디자인       |
| 넓은 의미 | 제품의 기획부터 생산, 판매까지 전 과정에 관여하는 포괄적 디자인 행위 |

**`제품 디자인 분류`**

| 분류                           | 설명                                                                    |
| ------------------------------ | ----------------------------------------------------------------------- |
| 생산 수단에 의한 분류          | 수공예 디자인(핸드크래프트)과 공업 디자인(인더스트리얼 디자인)으로 구분 |
| 도구적 장비의 성격에 의한 분류 | 가구, 가전제품, 수송 기기, 완구 등 사용 목적별로 구분                   |

### 환경 디자인(Environment Design)

환경 디자인이란 환경적인 문제인식을 가지며 사회와 자연의 관계를 통하여 쾌적한 환경 조성을 목적으로 하는 모든 디자인 분야를 포괄하는 상위 개념이다.

환경 디자인은 자연을 보호함으로써 인간을 보호하는 목적을 가진다.

**`환경 디자인 분야`**

| 분야                              | 설명                                                              |
| --------------------------------- | ----------------------------------------------------------------- |
| 도시 디자인                       | 도로, 광장, 공원 등 도시 공간 전체를 계획·설계하는 분야           |
| 실내(Interior) 디자인             | 건물 내부 공간의 기능성과 심미성을 높이기 위한 디자인 분야        |
| 디스플레이 디자인                 | 전시장, 쇼룸, 박람회 등 전시 공간을 연출하는 디자인 분야          |
| 스트리트 퍼니처(Street Furniture) | 가로등, 벤치, 버스 정류장 등 도시 시설물 디자인 분야              |
| 슈퍼 그래픽                       | 건물 외벽, 도로 등 대형 면적에 그래픽을 적용하는 환경 시각 디자인 |
