## 시안 개발계획 수립

### 스토리

데이터 간 연결, 정보 요소의 유형화, 체계화, 비교 등 다양한 구조를 시각화하는 작업.

| 스토리 구조        | 설명                                                                     | 특징                                                       |
| ------------------ | ------------------------------------------------------------------------ | ---------------------------------------------------------- |
| 선형적 스토리 구조 | 정보를 일렬로 순서대로 연결하는 구조. A → B → C처럼 하나의 흐름으로 진행 | 시작점과 끝점이 명확함. 시간 순서나 단계별 설명에 적합     |
| 위계적 스토리 구조 | 정보를 계층적(트리 구조)으로 조직화하는 구조. 전체 → 부분 방향으로 구성  | 사이트 맵·조직도 등에 활용. 정보의 상하 관계를 명확히 표현 |
| 유기적 스토리 구조 | 정보가 비선형적으로 상호 연결된 구조. 사용자가 자유롭게 탐색 가능        | 하이퍼텍스트·웹 구조가 대표적. 네트워크처럼 유연하게 연결  |

### 기호학적 측면의 이미지 유형

퍼스(Charles Sanders Peirce)의 기호론에 기반한 분류.

| 유형                          | 설명                                                                     | 특징                                           |
| ----------------------------- | ------------------------------------------------------------------------ | ---------------------------------------------- |
| 즉시적 이미지(Iconic Image)   | 대상과 시각적으로 유사한 이미지. 대상을 직접 닮은 형태로 표현            | 사진·그림·초상화 등. 아이콘(Icon)이라고도 함   |
| 상징적 이미지(Symbolic Image) | 대상과 직접 유사성이 없으며 사회적 약속·관습에 의해 의미가 부여된 이미지 | 국기·로고·화살표 등. 학습을 통해 의미를 이해함 |
| 지시적 이미지(Index Image)    | 대상과 인과관계·연관성을 가진 이미지. 대상의 존재를 간접적으로 암시함    | 연기(→ 불), 발자국(→ 사람), 온도계 등          |

### 내용과 표현

모든 이미지는 내용(Contents)과 표현(Expression)을 내포한다.

| 구분 | 설명                                                                                                         |
| ---- | ------------------------------------------------------------------------------------------------------------ |
| 내용 | 이미지가 전달하는 의미·메시지·주제. 이미지가 무엇을 말하고자 하는지에 해당하는 개념적 층위                   |
| 표현 | 이미지를 구성하는 시각적 형태·색상·질감·크기 등 조형 요소. 내용을 어떻게 시각화했는지에 해당하는 형식적 층위 |

## 아트워크 및 베리에이션

### 아트워크(Artwork)

아트워크란 디자인 콘셉트에 맞게 준비된 시각 자료를 활용하여 창의적으로 디자인 이미지를 만들어 내는 과정이다.

디자인 SW등을 활용한 레이아웃, 타이포그래픽 설정 등 모든 과정을 포함한다.

**`효과 배색`**

| 종류                           | 설명                                                                               | 특징                                                           |
| ------------------------------ | ---------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| 강조 배색(Accent)              | 전체 배색에 보색 또는 대비색을 소량 사용하여 시각적 포인트를 주는 배색             | 단조로운 배색에 활력을 부여하고 주목성을 높임                  |
| 분리 배색(Seperation)          | 대비가 강한 두 색 사이에 무채색(흰색·검정·회색) 등을 삽입하여 조화를 유도하는 배색 | 각 색을 명확히 분리하여 안정감을 줌                            |
| 그라데이션 배색(Gradation)     | 색상·명도·채도 등이 단계적으로 점진적 변화하는 배색                                | 자연스러운 흐름과 율동감·깊이감을 표현                         |
| 반복 배색(Repetition)          | 동일한 색상·색조·배색 패턴을 반복 사용하는 배색                                    | 리듬감과 통일성을 부여. 패턴 디자인·스트라이프 등에 활용       |
| 비콜로 배색(Bicolore)          | 두 가지 색상만을 사용하는 배색. 프랑스어로 '두 가지 색'                            | 명확하고 강한 인상을 줌. 국기 디자인 등에 활용                 |
| 토리콜로 배색(Tricolore)       | 세 가지 색상을 사용하는 배색. 프랑스어로 '세 가지 색'. 삼색기에서 유래             | 색상 대비를 통해 선명하고 생동감 있는 인상을 줌                |
| 톤 온 톤 배색(Tone on Tone)    | 동일 색상 계열에서 톤(명도·채도)을 변화시켜 조화를 이루는 배색                     | 차분하고 세련된 인상. 같은 색상의 밝기·채도 차이를 이용        |
| 톤 인 톤 배색(Tone in Tone)    | 다른 색상이지만 동일하거나 유사한 톤끼리 배색하는 방법                             | 은은하고 안정적인 인상. 색상은 다르지만 통일된 톤으로 조화     |
| 토널 배색(Tonal)               | 중간 채도의 탁한 톤 색들끼리 조합하는 배색                                         | 부드럽고 차분한 느낌. 자연스럽고 세련된 분위기                 |
| 까마이외 배색(Camaieu)         | 거의 동일한 색상·명도·채도를 가진 색들끼리 배색                                    | 통일감이 매우 강하고 은은함. 미묘한 차이만 존재                |
| 포 까마이외 배색(Faux Camaieu) | 까마이외보다 약간 더 폭넓은 색상 차이를 가진 배색. 프랑스어로 '가짜 까마이외'      | 전체적으로 통일감을 유지하면서 까마이외보다 약간의 변화를 허용 |

### BI(Brand Identity) 베이직 시스템 구성항목

| 구성 항목            | 설명                                                                                                            |
| -------------------- | --------------------------------------------------------------------------------------------------------------- |
| 심벌 마크            | 브랜드를 상징하는 그래픽 기호·도형. 텍스트 없이 시각적 이미지만으로 브랜드를 표현. (Nike 스우시, Apple 사과 등) |
| 로고 타입            | 브랜드명을 고유한 서체·디자인으로 표현한 문자형 로고. 텍스트만으로 구성. (Coca-Cola, Google 등)                 |
| 시그니처             | 심벌 마크와 로고 타입을 조합한 형태. 브랜드 아이덴티티의 대표 조합. 가로형·세로형 등 다양한 방식으로 구성       |
| 지정 컬러            | 브랜드를 대표하는 색상. 주색·보조색으로 구성하여 일관된 브랜드 인식도를 높임                                    |
| 지정 서체            | 브랜드에서 공식 사용하는 서체. 타이틀용·본문용으로 구분하여 지정. 일관된 타이포그래피 유지를 위함               |
| 그래픽 모티프(Motif) | 브랜드를 표현하는 반복적 패턴·형태·그래픽 요소. 배경·디자인 요소로 활용                                         |
| 캐릭터               | 브랜드를 대표하는 의인화된 캐릭터. 친근감을 높이고 브랜드 메시지를 전달                                         |

### Symbol 스타일

| 구분                        | 설명                                                             | 특징                                                        |
| --------------------------- | ---------------------------------------------------------------- | ----------------------------------------------------------- |
| 연결/오버랩 유형 스타일     | 두 개 이상의 형태가 겹치거나 연결되어 구성된 심벌 스타일         | 연결·통합·협력 등의 의미를 시각화. 연결고리·중첩 원 형태 등 |
| 심플/미니멀 유형 스타일     | 복잡한 요소를 제거하고 단순하고 명확한 형태로 표현한 심벌 스타일 | 현대적이고 세련된 인상. 다양한 크기에서 가독성·활용성 우수  |
| 유기적/다이나믹 유형 스타일 | 자연의 유기적 곡선이나 동적인 흐름을 표현한 심벌 스타일          | 생동감·움직임·유연함을 표현                                 |

### 레이아웃 베리에이션
