## 시안 개발계획 수립

### 스토리

데이터 간 연결, 정보 요소의 유형화, 체계화, 비교 등 다양한 구조를 시각화하는 작업.

| 스토리 구조        | 설명                                                                     | 특징                                                       |
| ------------------ | ------------------------------------------------------------------------ | ---------------------------------------------------------- |
| 선형적 스토리 구조 | 정보를 일렬로 순서대로 연결하는 구조. A → B → C처럼 하나의 흐름으로 진행 | 시작점과 끝점이 명확함. 시간 순서나 단계별 설명에 적합     |
| 위계적 스토리 구조 | 정보를 계층적(트리 구조)으로 조직화하는 구조. 전체 → 부분 방향으로 구성  | 사이트 맵·조직도 등에 활용. 정보의 상하 관계를 명확히 표현 |
| 유기적 스토리 구조 | 정보가 비선형적으로 상호 연결된 구조. 사용자가 자유롭게 탐색 가능        | 하이퍼텍스트·웹 구조가 대표적. 네트워크처럼 유연하게 연결  |

### 기호학적 측면의 이미지 유형

퍼스(Charles Sanders Peirce)의 기호론에 기반한 분류.

| 유형                          | 설명                                                                     | 특징                                           |
| ----------------------------- | ------------------------------------------------------------------------ | ---------------------------------------------- |
| 즉시적 이미지(Iconic Image)   | 대상과 시각적으로 유사한 이미지. 대상을 직접 닮은 형태로 표현            | 사진·그림·초상화 등. 아이콘(Icon)이라고도 함   |
| 상징적 이미지(Symbolic Image) | 대상과 직접 유사성이 없으며 사회적 약속·관습에 의해 의미가 부여된 이미지 | 국기·로고·화살표 등. 학습을 통해 의미를 이해함 |
| 지시적 이미지(Index Image)    | 대상과 인과관계·연관성을 가진 이미지. 대상의 존재를 간접적으로 암시함    | 연기(→ 불), 발자국(→ 사람), 온도계 등          |

### 내용과 표현

모든 이미지는 내용(Contents)과 표현(Expression)을 내포한다.

| 구분 | 설명                                                                                                         |
| ---- | ------------------------------------------------------------------------------------------------------------ |
| 내용 | 이미지가 전달하는 의미·메시지·주제. 이미지가 무엇을 말하고자 하는지에 해당하는 개념적 층위                   |
| 표현 | 이미지를 구성하는 시각적 형태·색상·질감·크기 등 조형 요소. 내용을 어떻게 시각화했는지에 해당하는 형식적 층위 |
