## 아이디어 발상 기법

심리학자 Joy Paul Guilford는 인간의 사고에는 **확산적 사고** 와 **수렴적 사고** 가 있으며, 창의적 사고는 확산과 수렴적 사고가 반복되면서 일어난다고 주장했다.

### 아이디어 발상 기법 구분

| 구분     | 특징                                                                          | 방법 종류                                                         |
| -------- | ----------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| 확산기법 | 기존 틀에서 벗어나 자유롭게 많은 아이디어를 생성하는 기법. 양적 접근          | 브레인스토밍, 브레인라이팅, 스캠퍼(SCAMPER), 마인드맵, 형태분석법 |
| 수렴기법 | 다양한 아이디어를 분석·평가하여 최선의 해결책을 선택·정리하는 기법. 질적 접근 | 어피니티 다이어그램, KJ법, 평가 행렬법                            |
| 통합기법 | 확산과 수렴을 반복·통합하여 창의적 해결책을 도출하는 기법                     | 시네틱스(Synectics), NM법, 트리즈(TRIZ)                           |

### 어피니티 다이어그램(Affinity Diagram)

**KJ법** 이라고도 불린다. (고안자 川喜田二郎의 이니셜)

브레인스토밍 등으로 도출한 많은 아이디어·데이터를 카드(포스트잇)에 개별 기록한 뒤, 유사한 것끼리 그룹화하여 **친화 관계** 를 발견하는 수렴기법.

복잡한 문제를 구조화하고 패턴을 발견하는 데 유용하며, 팀 전체가 참여하여 의견을 시각화할 수 있다.

### 무드보드(Mood Board)

디자인 콘셉트의 방향성·분위기·스타일을 시각적으로 표현하기 위해 이미지·색상·텍스트·패턴·소재 샘플 등을 하나의 보드에 수집·배치한 것.

프로젝트 초기 단계에서 팀원·클라이언트와 방향성을 공유하고 합의를 이끌어내기 위해 사용한다.

디지털(Photoshop 등) 또는 물리적 형태로 제작 가능하다.

## 아이디어 스캐치 및 비주얼 방향

### 아이디어 스캐치 방식 분류

| 분류              | 설명                                                                                          | 특징                                                                             |
| ----------------- | --------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| 개념 스케치       | 아이디어나 개념을 추상적·상징적으로 표현한 스케치. 형태보다 기능·관계·흐름을 표현하는 데 중점 | 마인드맵, 플로우 다이어그램 형태로 나타날 수 있음                                |
| 사물 스케치       | 제품·사물의 형태·구조·외관을 구체적으로 표현한 스케치                                         | 제품 디자인에서 형태·비율·세부 구조를 표현할 때 사용                             |
| 체험(경험) 스케치 | 사용자가 제품·서비스를 사용하는 경험·시나리오·상황을 표현한 스케치                            | UX 디자인에서 사용자 여정·행동·감정을 시각화. 스토리보드 형식으로 나타날 수 있음 |

### 비주얼 리터러시(Visual Literacy)

시각적 이미지와 정보를 읽고 이해하며, 생산하고 해석하는 능력을 Visual Literacy라고 한다.

1969년 John Debes가 처음 제안한 개념으로, 시각적 커뮤니케이션 능력을 의미한다.

| 구성 요소                      | 설명                                           |
| ------------------------------ | ---------------------------------------------- |
| 시각적 수용(Visual Reception)  | 이미지와 시각 정보를 인식하고 이해하는 능력    |
| 시각적 표현(Visual Expression) | 생각과 개념을 시각적으로 표현하는 능력         |
| 시각적 사고(Visual Thinking)   | 시각 정보를 통해 사고하고 문제를 해결하는 능력 |

디자이너에게 비주얼 리터러시는 색상·형태·타이포그래피·레이아웃 등 시각 요소를 이해하고 활용하여 효과적인 커뮤니케이션을 가능하게 하는 핵심 역량이다.

## 비주얼 아이데이션 적용

### 비주얼 펀(Visual Fun)

시각적 유머와 재미를 디자인에 적용하여 사용자에게 즐거운 경험을 제공하는 디자인 접근 방식.

디자인에 유머·위트·놀이성을 부여하여 사용자와의 감성적 연결을 강화하고, 기억에 남는 경험을 만든다.

예상치 못한 시각적 요소의 배치, 익살스러운 캐릭터, 인터랙티브 요소 등을 통해 구현된다.

### 사용자 편의를 위한 디자인 가치

| 디자인 가치                         | 설명                                                                                          | 특징                                                                  |
| ----------------------------------- | --------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| 유니버셜 디자인(Universal Design)   | 연령·성별·장애 여부에 관계없이 모든 사람이 사용할 수 있도록 설계하는 디자인. Ron Mace가 제안  | 공평성·유연성·단순성·낮은 신체적 노력 등 7원칙에 기반                 |
| 지속가능 디자인(Sustainable Design) | 환경적·사회적·경제적 지속가능성을 고려한 디자인. 자원 낭비 최소화, 환경 영향 감소를 목표로 함 | 재활용 가능한 소재 사용, 에너지 효율성 고려                           |
| 어메니티 디자인(Amenity Design)     | 생활 환경의 쾌적함·편의성·아름다움을 높이는 디자인                                            | 도시 환경·공공장소·생활 용품 등을 사용자에게 쾌적하고 편리하도록 설계 |

### 일러스트레이션 유형

| 유형                    | 설명                                                                                | 특징                                                         |
| ----------------------- | ----------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| 추상적 일러스트레이션   | 구체적 형상 없이 색·형·선·질감 등 조형 요소만으로 감정·개념을 표현                  | 사실적 표현보다 감각적·감정적 전달에 중점                    |
| 구상적 일러스트레이션   | 실제 존재하는 사물·인물·자연 등을 구체적으로 묘사                                   | 광고·교과서·만화 등에 주로 사용. 사실적 또는 단순화하여 표현 |
| 초현실적 일러스트레이션 | 현실에서 존재하지 않는 비논리적·초자연적 상황을 표현. 초현실주의(Surrealism)에 기반 | 꿈·환상·상상의 세계를 시각화                                 |

**`초현실적 일러스트레이션 유형`**

| 유형                    | 설명                                                                                            | 특징                                                |
| ----------------------- | ----------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| 몽타주(Montage)         | 서로 다른 이미지·필름 조각을 이어 붙여 새로운 의미를 만드는 기법. 영상·사진에서 유래            | 개별 이미지와 다른 새로운 맥락·의미를 창출          |
| 더페이즈망(Depaysement) | 익숙한 사물을 전혀 다른 맥락·환경에 배치하여 낯선 느낌을 주는 기법. 프랑스어로 "낯선 곳에 놓다" | 르네 마그리트의 작품에서 대표적으로 나타남          |
| 콜라주(Collage)         | 신문·잡지·사진·천 등 다양한 재료를 오려 붙여 새로운 이미지를 만드는 기법                        | 브라크·피카소에 의해 발전. 디지털 콜라주로도 활용됨 |
