## 색의 기본 원리

### 물체의 색

물체의 색은 표면색, 투과색, 광원색, 간섭색 등으로 구분된다.

**`표면색`**

빛이 불투명 물체의 표면에서 반사되어 눈에 지각되는 색. 일상에서 가장 흔하게 인식되는 물체의 색.

| 종류           | 설명                                                           |
| -------------- | -------------------------------------------------------------- |
| 물체색(반사색) | 불투명 물체 표면의 반사에 의한 색. 표면색의 가장 일반적인 형태 |
| 경면색         | 거울·금속 등 광택면에서 빛이 정반사(거울 반사)되어 나타나는 색 |
| 형광색         | 자외선 등 특정 파장을 흡수하여 더 밝은 가시광선을 방출하는 색  |

**`투과색`**

투명하거나 반투명한 물체(유리, 셀로판 등)를 빛이 통과할 때 보이는 색. 특정 파장의 빛이 선택적으로 투과되어 색이 나타남.

| 종류               | 설명                                                                     |
| ------------------ | ------------------------------------------------------------------------ |
| 공간색(Film Color) | 색유리나 맑은 하늘처럼 색이 3차원 공간에 가득 차 있는 것처럼 지각되는 색 |
| 반투과색           | 반투명 물체(색 셀로판 등)를 통과하면서 일부 파장이 흡수·산란된 색        |

**`광원색`**

태양, 전구, 모니터 등 빛을 스스로 발하는 광원이 가진 색. 물체에 반사·투과되는 것이 아닌 광원 자체의 발광에 의한 색으로, 가산혼합의 원리가 적용된다.

| 종류       | 설명                                                     |
| ---------- | -------------------------------------------------------- |
| 자연광원색 | 태양, 불꽃 등 자연에서 발생하는 광원의 색                |
| 인공광원색 | 전구, LED, 형광등, 모니터 등 인공 조명 및 발광 장치의 색 |

**`간섭색`**

빛의 간섭(Interference) 현상에 의해 나타나는 색. 물체 자체에 색소가 있는 것이 아니라, 빛이 얇은 막의 표면과 뒷면에서 반사될 때 파장 간 보강·상쇄 간섭으로 발생한다.

비눗방울, 기름막, 나비 날개의 비늘(구조색) 등에서 관찰할 수 있다.

### 색채 자극과 반응

**`명암 / 색순응`**

| 구분   | 설명                                                                      |
| ------ | ------------------------------------------------------------------------- |
| 명순응 | 어두운 곳에서 밝은 곳으로 이동 시, 눈이 밝은 환경에 적응하는 현상         |
| 암순응 | 밝은 곳에서 어두운 곳으로 이동 시, 눈이 어두운 환경에 적응하는 현상       |
| 색순응 | 특정 색광 아래에 오래 있으면 그 색에 적응하여 중성색처럼 느끼게 되는 현상 |

암순응 속도보다 명순응 속도가 빠르다.

**`연색성과 조건등색`**

| 구분     | 설명                                                                                                    |
| -------- | ------------------------------------------------------------------------------------------------------- |
| 연색성   | 광원이 물체의 색을 어떻게 보이게 하는지에 대한 광원의 특성. 연색지수(CRI)로 측정하며 태양광이 기준(100) |
| 조건등색 | 서로 다른 분광 반사율을 가진 두 색이 특정 광원 아래에서 같은 색으로 보이는 현상 (메타메리즘)            |

**`색각 이상`**

| 구분 | 설명                                                                              |
| ---- | --------------------------------------------------------------------------------- |
| 색맹 | 색각 이상으로 특정 색을 전혀 구별하지 못하는 상태. 적록색맹이 가장 흔함           |
| 색약 | 색을 구별하는 능력이 정상보다 약한 상태. 색맹보다 경증으로 어느 정도 색 구별 가능 |

### 색의 지각효과

| 효과                      | 설명                                                                                                        |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- |
| 색음(Colored Shadow) 효과 | 색이 있는 광원 아래 그림자에 보색이 나타나는 현상. 예) 붉은 조명 아래 그림자가 녹색으로 보임                |
| 애브니(Abney) 효과        | 색상이 동일해도 채도가 변하면 색조(Hue)가 달라 보이는 현상. 같은 색에 흰색이 더해지면 색상 자체가 달라 보임 |
| 베졸드(Bezold) 효과       | 소량의 색 변경이 전체 색 배합 인상을 바꾸는 현상. 색의 확산효과라고도 함                                    |
| 맥칼로(McCollough) 효과   | 색상-방향 패턴을 오래 바라본 후 흑백 패턴을 보면 보색 잔상이 나타나는 현상                                  |

### 색채 지각설

**`Young / Helmholtz의 3원색설`**

망막에 R(빨강)·G(초록)·B(파랑)에 대응하는 세 종류의 수용체가 있으며, 이 수용체들의 자극 비율로 모든 색이 인지된다는 이론.

| 수용체   | 대응 파장 | 감지 색 |
| -------- | --------- | ------- |
| L 수용체 | 장파장    | 빨강(R) |
| M 수용체 | 중파장    | 초록(G) |
| S 수용체 | 단파장    | 파랑(B) |

- 가산혼합(빛의 혼합) 원리를 잘 설명하며 색맹 현상도 설명 가능
- 단점: 보색 잔상·동시 대비 등 일부 현상은 설명하기 어려움

**`Hering의 반대색설(4원색설)`**

색각은 R-G(빨강-초록), Y-B(노랑-파랑), W-Bk(흰-검정)의 세 쌍 반대색 수용체가 대립하는 방식으로 색을 인지한다는 이론.

기본색을 R·G·Y·B 4가지로 본다.

| 반대색 쌍   | 설명                               |
| ----------- | ---------------------------------- |
| 빨강 - 초록 | 한 쪽이 활성화되면 반대쪽이 억제됨 |
| 노랑 - 파랑 | 한 쪽이 활성화되면 반대쪽이 억제됨 |
| 흰색 - 검정 | 밝기 대립 채널                     |

- 보색 잔상·동시 대비·색순응 현상을 잘 설명
- 단점: 망막 수용체 단계의 물리적 메커니즘 설명이 불충분

**`혼합설`**

3원색설과 반대색설을 단계적으로 통합한 이론. Zone Theory(존 이론)라고도 한다.

| 처리 단계                | 적용 이론               | 설명                                                |
| ------------------------ | ----------------------- | --------------------------------------------------- |
| 1단계 — 망막 수용체 단계 | Young-Helmholtz 3원색설 | R·G·B 세 수용체가 빛을 수용하여 신호를 생성         |
| 2단계 — 신경 처리 단계   | Hering 반대색설         | 세 쌍의 반대색 채널로 신호를 재부호화하여 뇌로 전달 |

두 이론의 장점을 결합하여 대부분의 색각 현상을 통합적으로 설명할 수 있다.

## 색의 혼합 및 표시방법

### 표색계

| 구분   | 설명                                                                  | 특징                                                                     | 대표 예시                                     |
| ------ | --------------------------------------------------------------------- | ------------------------------------------------------------------------ | --------------------------------------------- |
| 현색계 | 인간이 색을 지각하는 방식(색상·명도·채도)을 기준으로 색을 배열한 체계 | 실물 색표(Color Chip) 기반. 지각적 균등성을 추구하며 직관적 색 분류 가능 | 먼셀(Munsell), 오스트발트(Ostwald), KS 색체계 |
| 혼색계 | 빛의 혼합(스펙트럼) 원리를 기반으로 색을 수치화하는 체계              | 색자극값(X, Y, Z)으로 색을 수량 표현. 기기 측색·색 재현 관리에 적합      | CIE XYZ, CIE Lab, CIE Luv                     |

### 오스트발트(Ostwald) 색체계

헤링의 4원색설(RGBY)을 기본으로 한다.

**`색상환`**

4색 사이 중간색을 넣어 8색으로 구분하고, 이러한 방식으로 추가로 3등분하여 24색상으로 구분한다.

명도는 8단계로 나눠 사용한다.

**`혼합비`**

혼합비는 W(White) / B(Black) / C(Color) 세 변수를 사용하며, 세 변수의 합은 항상 100%를 유지한다.

- 무채색은 C값이 0이므로, W + B = 100%

- 유채색은 C + W + B = 100%

**`색체계 기호 표기법`**

8 x 8 평면을 마름모형태로 배치하고, 위 꼭짓점 부터 밑 꼭짓점을 가로질러 8단계의 명도(a, c, e, g, i, l, n, p) 기호를 둔다.

- a는 가장 밝은(White)색상이며, p는 가장 어두운(Black) 색상이다.

## 색의 지각과 심리

### 색의 시간성

미국의 색채 연구가 Faber Birren은 색 파장에 따른 시간성의 차이를 주장했다.

| 구분                    | 효과                        | 활용 예시                                                                                   |
| ----------------------- | --------------------------- | ------------------------------------------------------------------------------------------- |
| 장파장 계통의 난색 계열 | 시간이 실제보다 길게 느껴짐 | 패스트푸드·음식점 등 회전율을 높이고 싶을 때. 시간이 길게 느껴져 손님이 빨리 자리를 뜨게 됨 |
| 단파장 계통의 한색 계열 | 시간이 실제보다 짧게 느껴짐 | 병원 대기실 등 기다리는 시간이 짧게 느껴지게 하고 싶은 공간에 활용                          |

### 색의 공감각

색을 지각할 때, 시각외에 다른 감각까지 느껴지는 특성을 공감각이라고 한다.

| 구분 | 특징                                          | 예시                                                  |
| ---- | --------------------------------------------- | ----------------------------------------------------- |
| 청각 | 색의 밝기·채도에 따라 음의 높낮이·크기를 느낌 | 밝은 색 → 고음·경쾌함, 어두운 색 → 저음·무거움        |
| 미각 | 색을 보고 단맛·신맛·쓴맛 등 맛을 연상함       | 빨강·주황 → 단맛, 노랑 → 신맛, 초록·갈색 → 쓴맛       |
| 후각 | 색을 보고 특정 향기나 냄새를 연상함           | 보라 → 라벤더 향, 초록 → 풀냄새, 노랑 → 상큼한 과일향 |

### 색의 연상

색을 지각할 때, 개인의 경험과 심리적 작용을 기반으로 부수적인 개념을 떠올리는 특징을 색의 연상이라고 한다.

| 구분        | 설명                                                         | 예시                                 |
| ----------- | ------------------------------------------------------------ | ------------------------------------ |
| 추상적 연상 | 색을 보고 감정·개념·분위기 등 추상적인 것을 떠올리는 연상    | 빨강 → 열정·위험, 파랑 → 차가움·신뢰 |
| 구체적 연상 | 색을 보고 실제 사물·자연 현상 등 구체적 대상을 떠올리는 연상 | 파랑 → 하늘·바다, 초록 → 나무·풀     |

## 색채조화

### 유사조화

색상환에서 인접한 색상들이 배색되었을 때 잘 어울리는 현상.

| 구분                   | 설명                                                                 | 예시                           |
| ---------------------- | -------------------------------------------------------------------- | ------------------------------ |
| 색상의 조화            | 색상환에서 인접한 유사 색상끼리 배색. 부드럽고 자연스러운 느낌       | 노랑 + 연두 + 초록             |
| 단계적 조화(명도 조화) | 동일 색상 내에서 명도를 단계적으로 변화시킨 배색. 통일감과 리듬감    | 연파랑 → 파랑 → 진파랑         |
| 주조색의 조화          | 주조색을 중심으로 보조색·강조색을 유사 계열로 배색. 일관성 있는 인상 | 파란 계열 주조색 + 청록 보조색 |

### 대비조화

색상환에서 반대색이 배색되었을 때 잘 어울리는 현상.

| 구분     | 설명                                                                | 예시                         |
| -------- | ------------------------------------------------------------------- | ---------------------------- |
| 색상조화 | 색상환에서 서로 거리가 먼 색상끼리 배색. 강렬하고 생동감 있는 효과  | 빨강 + 파랑, 노랑 + 보라     |
| 명도조화 | 밝은 색과 어두운 색의 대비를 활용한 배색. 명확한 시각적 대비        | 흰색 + 검정, 연노랑 + 진남색 |
| 보색조화 | 색상환에서 정반대에 위치한 보색끼리 배색. 서로를 더욱 선명하게 강화 | 빨강 + 초록, 파랑 + 주황     |

## 제도일반

### 제도의 정의

제도란 도면을 통해 작성자의 의도를 쉽게 전달하기 위한 방법을 말한다.

입체 형태인 물체의 크기 또는 모양을 일정한 규격과 규칙에 따라 평면 위에 점, 선, 문자, 부호 등을 이용하여 도면에 작성하는 것을 말한다.

**`제도 문자`**

도면에 기입되는 문자를 제도 문자라고 한다.

```text
KS A 0005 (제도 문자)

KS : 규격
A : 부문 기호
0005 : 일련 번호
```

**`제도 규격`**

| 규격기호 | 명칭                                                              |
| -------- | ----------------------------------------------------------------- |
| ISO      | International Organization for Standardization (국제 표준화 기구) |
| KS       | Korean Standards (한국 산업 표준)                                 |
| ANSI     | American National Standards Institute (미국 국가 표준 협회)       |
| BS       | British Standards (영국 규격)                                     |
| DIN      | Deutsches Institut für Normung (독일 규격)                        |
| JIS      | Japanese Industrial Standards (일본 공업 규격)                    |

**`부문 기호`**

KS 규격에서 해당 규격이 속하는 분야를 나타내는 알파벳 기호.

- 예: KS **A** 0005에서 A가 부문 기호.

| 부문 기호 | 분야                          |
| --------- | ----------------------------- |
| A         | 기본(Basic)                   |
| B         | 기계(Mechanical)              |
| C         | 전기·전자(Electrical)         |
| D         | 금속(Metals)                  |
| F         | 건설(Construction)            |
| K         | 화학(Chemistry)               |
| L         | 섬유(Textile)                 |
| Q         | 품질 경영(Quality Management) |
| X         | 정보산업(IT)                  |

**`일련 번호`**

해당 부문 내에서 개별 규격을 구별하기 위한 4자리 숫자.

- 예: KS A **0005**에서 0005가 일련 번호.

### 도면의 종류

| 종류             | 설명                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------- |
| 시방서           | 도면만으로 표현하기 어려운 재료·공법·주의사항 등을 글로 기술한 문서. 설계 도서의 일종 |
| 공정도           | 제조·시공 과정의 순서와 방법을 단계별로 나타낸 도면                                   |
| 입면도           | 건물이나 구조물의 외관을 정면·측면 등 수직 방향에서 바라본 투영도                     |
| 원도             | 실제 크기나 일정 축척으로 작성된 최초의 원본 도면                                     |
| 사도(트레이스도) | 원도를 트레이싱 페이퍼 위에 겹쳐 베껴 그린 복사 도면                                  |

### 제도용지

**`도면의 크기`**

도면 크기 비율은 $1 : \sqrt{2}$ 이며, mm단위로 표시한다.

제도 용지의 크기는 A0, A1 ... 으로 구분된다.

- A4: 210mm &times; 297mm

**`제도용지 종류`**

| 종류       | 설명                                                                          |
| ---------- | ----------------------------------------------------------------------------- |
| 켄트지     | 하얗고 단단한 불투명 용지. 잉크·연필 모두 적합하며 일반 제도에 가장 많이 사용 |
| 와트만지   | 두껍고 거친 수채화 전용 용지. 수채화 표현에 적합                              |
| 트레이싱지 | 반투명 용지. 원도 위에 겹쳐 베껴 그리는 사도(트레이스도) 제작에 사용          |
| 방안지     | 일정 간격의 격자(눈금)가 인쇄된 용지. 그래프·도면 초안 작성에 사용            |

**`도면의 척도`**

물체를 도면에 표기할 때 실제 크기에 대한 도면의 크기 비율을 척도라고 한다.

척도는 **표제란** 에 기입한다.

- 표제란은 도면에 오른쪽 맨 아래 표로 위치하며, 도면 관리에 필요한 정형적인 사항을 표함한다.

| 종류 | 설명                                                                              |
| ---- | --------------------------------------------------------------------------------- |
| 실척 | 실제 크기와 동일하게 그린 도면. 비율 1:1                                          |
| 축적 | 실제 크기보다 작게 줄여 그린 도면. 비율 1:n (n>1). 큰 물체 표현에 사용            |
| 배척 | 실제 크기보다 크게 확대하여 그린 도면. 비율 n:1 (n>1). 작은 부품·세부 표현에 사용 |
| N.S  | Not to Scale. 척도 없이 그린 도면. 참고용·개략도에 사용                           |

### 제도 선 종류

| 종류                             | 명칭 및 용도                                                                                                                                                                       |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 굵은 실선                        | `외형선:` 물체의 보이는 겉모양 윤곽선                                                                                                                                              |
| 가는 실선                        | `치수선`: 치수를 기입하기 위한 선<br>`치수보조선`: 치수선 끝점에서 인출하는 선<br>`지시선`: 기호·주석 등을 가리키는 선<br>`회전 단면선`: 단면 형상을 90° 회전하여 외형선 위에 표시 |
| 굵은 건의 1/2파선                | `숨은선`: 물체의 보이지 않는 부분의 윤곽선                                                                                                                                         |
| 가는 일점 쇄선                   | `중심선`: 도형의 중심 위치를 나타내는 선<br>`기준선`: 치수·위치의 기준이 되는 선<br>`피치선`: 기어·볼트 등의 피치를 나타내는 선                                                    |
| 가는 이점 쇄선                   | `가상선`: 가공 전후 형상·인접 부품·이동 한계 등 참고 형상을 나타내는 선<br>`무게중심선`: 단면의 무게중심 궤적을 나타내는 선                                                        |
| 파형, 지그재그의 가는 실선       | `파단선`: 물체의 일부를 절단한 경계를 나타내는 선                                                                                                                                  |
| 가는 일점 쇄선, 변하는 부분 굵게 | `절단선`: 단면도에서 물체를 절단하는 위치를 나타내는 선                                                                                                                            |
| 45° 방향의 가는 실선             | `해칭선(Hatching Line)`: 단면도에서 절단면을 표시하는 선                                                                                                                           |

### 제도 기호

| 명칭               | 기호    | 용도                                                                  |
| ------------------ | ------- | --------------------------------------------------------------------- |
| 지름               | φ       | 원의 지름을 나타낼 때 치수 숫자 앞에 기입                             |
| 반지름             | R       | 원호의 반지름을 나타낼 때 치수 숫자 앞에 기입                         |

| 구의 지름 / 반지름 | Sφ / SR | 구의 지름(Sφ)과 구의 반지름(SR)을 나타낼 때                           |

## 투상도법

투상도법은 3차원 표현 대상을 2차원 평면상에 일정한 도법을 적용하여 표현하기 위해 사용하는 방법이다.

시선을 통해 들어오는 빛을 투상선이라고 하며, 투상선은 투상면을 통과하여 2차원 상이 표현된다.

투상면은 일반적으로 특수한 유리면으로 되어있는데, 투상면이 필요한 이유는 투상선을 일정한 각도(직각)으로 잡아주는 역할을 하기 때문이다. 따라서 3차원 물체의 크기를 정확히 파악할 수 있다.

### 투상의 종류

| 종류      | 설명                                                                               |
| --------- | ---------------------------------------------------------------------------------- |
| 직 투상   | 투상선이 투상면에 수직으로 만나는 방식. 정확한 치수 표현 가능 (정투상)             |
| 사 투상   | 투상선이 투상면에 비스듬한 각도로 만나는 방식. 앞면은 실형, 측면은 경사지게 표현   |
| 투시 투상 | 소실점으로 투상선이 수렴하는 방식. 실제 눈에 보이는 원근감 표현 (1점·2점·3점 투시) |
| 복 투상   | 여러 투상면에 동시에 투상하여 정면도·평면도·측면도 등 다수 뷰를 함께 표현하는 방식 |
| 단면 투상 | 물체를 가상으로 절단하여 내부 구조를 보이도록 투상하는 방식                        |

### 정 투상도

평화면(앞 / 뒤), 입화면(위 / 밑), 측화면(양옆) 사이에 물체를 놓고 각 화면에 수직되는 평행광선으로 투상하여 얻은 도형을 정투상도라고 한다.

평화면: 정면도(앞) / 배면도(뒤)
입화면: 평면도(위) / 저면도(밑)
측화면: 좌측면도(좌) / 우측면도(우)

**`제 3각법`**

한국산업규격에서 가장 많이 사용되는 정투상법이다.

눈(시선 / 투상선) -> 화면(투상면) -> 물체 순서로 화면을 통해서 물체를 보며, 보는 위치면에 따라서 상이 나타난다.

**`제 1각법`**

눈(시선 / 투상선) -> 물체 -> 화면(투상면) 순서로 진행되어, 투상면이 물체 뒤에 있는 방식의 정투상법이다.

### 축측 투상도

평화면, 입화면, 측화면이 모두 보이도록 물체의 각도를 결정하여 투상하는 방법이다.
각도에 따라 등각(Isometric), 2등각(Dimetric), 부등각(Trimetric) 투상으로 구분한다.

### 사 투상도

앞면(정화면) 모서리는 수평선과 평행하게 하고, 옆면(측화면) 모서리는 수평선과 임의의 각도로 하여 그린 투상도.

일반적으로 30°, 45°, 60° 의 각도를 사용함.

축측 투상도와 같이 정육면체의 세 면이 동시에 나타난다.

### 표고 투상도

기준면을 정한 후 기준 면과 평행하게 평면으로 자른 수평면을 수직으로 투상하여 그린 수직 투상도.

일반적으로 지형도를 그릴 때 사용된다.

## 투시도법

투시도법은 물체와 눈 사이에 투명한 화면을 놓고 시점과 물체를 연결한 투사선에 의해 비치는 현상을 그리는 방법을 말한다.

투시도법은 사람의 눈으로 보는 것과 같이 먼 곳에 있는 물체는 작게, 가까운 곳의 물체는 크게 그리는 도법으로, **원근 화법** 이라고도 한다.

투시도법은 레오나르도 다빈치가 최초로 연구하였고, 르네상스 시대에 발전하였다.

### 투시도법 원리

투시도는 시선을 중심으로 약 60° 범위 내의 방사선상이며, 시점의 위치에 따라 상이 다르게 그려진다.

### 투시도법 부호와 용어

| 부호 | 용어                     | 해설                                                      |
| ---- | ------------------------ | --------------------------------------------------------- |
| E.L  | 눈높이선(Eye Level)      | 관측자의 눈높이에서 그은 수평선. 소실점이 위치하는 기준선 |
| G.L  | 기선(Ground Line)        | 화면과 지면이 교차하는 선                                 |
| H.L  | 수평선(Horizon Line)     | E.L과 동일 개념으로 사용. 소실점이 놓이는 선              |
| G.P  | 지면(Ground Plane)       | 관측자가 위치하는 수평 기준면                             |
| P.P  | 화면(Picture Plane)      | 눈과 물체 사이에 놓인 투명한 투상면                       |
| S.P  | 시점(Station Point)      | 관측자(눈)의 위치                                         |
| V.P  | 소실점(Vanishing Point)  | 평행선들이 수렴하는 점. H.L 위에 위치                     |
| C.V  | 시심점(Center of Vision) | 시점에서 화면에 수직으로 내린 발점. 주소점이라고도 함     |
| M.P  | 계측점(Measure Point)    | 깊이(원근감) 측정을 위한 보조점                           |

### 투시도법 종류

| 종류         | 별칭                     | 설명                                                                                |
| ------------ | ------------------------ | ----------------------------------------------------------------------------------- |
| 1소점 투시도 | 평행 투시도              | 소실점 1개. 물체의 한 면이 화면과 평행. 실내 투시도에 주로 사용                     |
| 2소점 투시도 | 유각 투시도, 성각 투시도 | 소실점 2개. 물체의 모서리가 화면과 각도를 이루는 상태. 건물 외관 표현에 주로 사용   |
| 3소점 투시도 | 사각 투시도              | 소실점 3개. 물체를 위에서 내려다보거나 아래서 올려다볼 때. 고층 빌딩 등 표현에 사용 |
