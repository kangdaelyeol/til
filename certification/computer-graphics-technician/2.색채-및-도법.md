## 색의 기본 원리

### 물체의 색

물체의 색은 표면색, 투과색, 광원색, 간섭색 등으로 구분된다.

**`표면색`**

빛이 불투명 물체의 표면에서 반사되어 눈에 지각되는 색. 일상에서 가장 흔하게 인식되는 물체의 색.

| 종류           | 설명                                                           |
| -------------- | -------------------------------------------------------------- |
| 물체색(반사색) | 불투명 물체 표면의 반사에 의한 색. 표면색의 가장 일반적인 형태 |
| 경면색(경영색) | 거울·금속 등 광택면에서 빛이 정반사(거울 반사)되어 나타나는 색 |
| 형광색         | 자외선 등 특정 파장을 흡수하여 더 밝은 가시광선을 방출하는 색  |

**`투과색`**

투명하거나 반투명한 물체(유리, 셀로판 등)를 빛이 통과할 때 보이는 색. 특정 파장의 빛이 선택적으로 투과되어 색이 나타남.

| 종류               | 설명                                                                     |
| ------------------ | ------------------------------------------------------------------------ |
| 공간색(Film Color) | 색유리나 맑은 하늘처럼 색이 3차원 공간에 가득 차 있는 것처럼 지각되는 색 |
| 반투과색           | 반투명 물체(색 셀로판 등)를 통과하면서 일부 파장이 흡수·산란된 색        |

**`광원색`**

태양, 전구, 모니터 등 빛을 스스로 발하는 광원이 가진 색. 물체에 반사·투과되는 것이 아닌 광원 자체의 발광에 의한 색으로, 가산혼합의 원리가 적용된다.

| 종류       | 설명                                                     |
| ---------- | -------------------------------------------------------- |
| 자연광원색 | 태양, 불꽃 등 자연에서 발생하는 광원의 색                |
| 인공광원색 | 전구, LED, 형광등, 모니터 등 인공 조명 및 발광 장치의 색 |

**`간섭색`**

빛의 간섭(Interference) 현상에 의해 나타나는 색. 물체 자체에 색소가 있는 것이 아니라, 빛이 얇은 막의 표면과 뒷면에서 반사될 때 파장 간 보강·상쇄 간섭으로 발생한다.

비눗방울, 기름막, 나비 날개의 비늘(구조색) 등에서 관찰할 수 있다.

### 색채 자극과 반응

**`명암 / 색순응`**

| 구분   | 설명                                                                      |
| ------ | ------------------------------------------------------------------------- |
| 명순응 | 어두운 곳에서 밝은 곳으로 이동 시, 눈이 밝은 환경에 적응하는 현상         |
| 암순응 | 밝은 곳에서 어두운 곳으로 이동 시, 눈이 어두운 환경에 적응하는 현상       |
| 색순응 | 특정 색광 아래에 오래 있으면 그 색에 적응하여 중성색처럼 느끼게 되는 현상 |

암순응 속도보다 명순응 속도가 빠르다.

**`연색성과 조건등색`**

| 구분     | 설명                                                                                                    |
| -------- | ------------------------------------------------------------------------------------------------------- |
| 연색성   | 광원이 물체의 색을 어떻게 보이게 하는지에 대한 광원의 특성. 연색지수(CRI)로 측정하며 태양광이 기준(100) |
| 조건등색 | 서로 다른 분광 반사율을 가진 두 색이 특정 광원 아래에서 같은 색으로 보이는 현상 (메타메리즘)            |

**`색각 이상`**

| 구분 | 설명                                                                              |
| ---- | --------------------------------------------------------------------------------- |
| 색맹 | 색각 이상으로 특정 색을 전혀 구별하지 못하는 상태. 적록색맹이 가장 흔함           |
| 색약 | 색을 구별하는 능력이 정상보다 약한 상태. 색맹보다 경증으로 어느 정도 색 구별 가능 |

### 색의 지각효과

| 효과                      | 설명                                                                                                        |
| ------------------------- | ----------------------------------------------------------------------------------------------------------- |
| 색음(Colored Shadow) 효과 | 색이 있는 광원 아래 그림자에 보색이 나타나는 현상. 예) 붉은 조명 아래 그림자가 녹색으로 보임                |
| 애브니(Abney) 효과        | 색상이 동일해도 채도가 변하면 색조(Hue)가 달라 보이는 현상. 같은 색에 흰색이 더해지면 색상 자체가 달라 보임 |
| 베졸드(Bezold) 효과       | 소량의 색 변경이 전체 색 배합 인상을 바꾸는 현상. 색의 확산효과라고도 함                                    |
| 맥칼로(McCollough) 효과   | 색상-방향 패턴을 오래 바라본 후 흑백 패턴을 보면 보색 잔상이 나타나는 현상                                  |

### 색채 지각설

**`Young / Helmholtz의 3원색설`**

망막에 R(빨강)·G(초록)·B(파랑)에 대응하는 세 종류의 수용체가 있으며, 이 수용체들의 자극 비율로 모든 색이 인지된다는 이론.

| 수용체   | 대응 파장 | 감지 색 |
| -------- | --------- | ------- |
| L 수용체 | 장파장    | 빨강(R) |
| M 수용체 | 중파장    | 초록(G) |
| S 수용체 | 단파장    | 파랑(B) |

- 가산혼합(빛의 혼합) 원리를 잘 설명하며 색맹 현상도 설명 가능
- 단점: 보색 잔상·동시 대비 등 일부 현상은 설명하기 어려움

**`Hering의 반대색설(4원색설)`**

색각은 R-G(빨강-초록), Y-B(노랑-파랑), W-Bk(흰-검정)의 세 쌍 반대색 수용체가 대립하는 방식으로 색을 인지한다는 이론.

기본색을 R·G·Y·B 4가지로 본다.

| 반대색 쌍   | 설명                               |
| ----------- | ---------------------------------- |
| 빨강 - 초록 | 한 쪽이 활성화되면 반대쪽이 억제됨 |
| 노랑 - 파랑 | 한 쪽이 활성화되면 반대쪽이 억제됨 |
| 흰색 - 검정 | 밝기 대립 채널                     |

- 보색 잔상·동시 대비·색순응 현상을 잘 설명
- 단점: 망막 수용체 단계의 물리적 메커니즘 설명이 불충분

**`혼합설`**

3원색설과 반대색설을 단계적으로 통합한 이론. Zone Theory(존 이론)라고도 한다.

| 처리 단계                | 적용 이론               | 설명                                                |
| ------------------------ | ----------------------- | --------------------------------------------------- |
| 1단계 — 망막 수용체 단계 | Young-Helmholtz 3원색설 | R·G·B 세 수용체가 빛을 수용하여 신호를 생성         |
| 2단계 — 신경 처리 단계   | Hering 반대색설         | 세 쌍의 반대색 채널로 신호를 재부호화하여 뇌로 전달 |

두 이론의 장점을 결합하여 대부분의 색각 현상을 통합적으로 설명할 수 있다.

## 색의 혼합 및 표시방법

### 표색계

| 구분   | 설명                                                                  | 특징                                                                     | 대표 예시                                     |
| ------ | --------------------------------------------------------------------- | ------------------------------------------------------------------------ | --------------------------------------------- |
| 현색계 | 인간이 색을 지각하는 방식(색상·명도·채도)을 기준으로 색을 배열한 체계 | 실물 색표(Color Chip) 기반. 지각적 균등성을 추구하며 직관적 색 분류 가능 | 먼셀(Munsell), 오스트발트(Ostwald), KS 색체계 |
| 혼색계 | 빛의 혼합(스펙트럼) 원리를 기반으로 색을 수치화하는 체계              | 색자극값(X, Y, Z)으로 색을 수량 표현. 기기 측색·색 재현 관리에 적합      | CIE XYZ, CIE Lab, CIE Luv                     |

### 오스트발트(Ostwald) 색체계

헤링의 4원색설(RGBY)을 기본으로 한다.

**`색상환`**

4색 사이 중간색을 넣어 8색으로 구분하고, 이러한 방식으로 추가로 3등분하여 24색상으로 구분한다.

명도는 8단계로 나눠 사용한다.

**`혼합비`**

혼합비는 W(White) / B(Black) / C(Color) 세 변수를 사용하며, 세 변수의 합은 항상 100%를 유지한다.

- 무채색은 C값이 0이므로, $W + B = 100%$%

- 유채색은 $C + W + B = 100$%

**`색체계 기호 표기법`**

8 x 8 평면을 마름모형태로 배치하고, 위 꼭짓점 부터 밑 꼭짓점을 가로질러 8단계의 명도(a, c, e, g, i, l, n, p) 기호를 둔다.

- a는 가장 밝은(White)색상이며, p는 가장 어두운(Black) 색상이다.
