## 웹디자인 시스템 및 인력 구성

### 웹디자인

웹디자인(Web Design)이란 웹사이트를 설계하고 디자인하는 것이다.

콘텐츠를 심미적으로 균형을 이루어 배치하여 효율적인 UI/UX를 구축한다.

**`웹디자인 프로세스`**

| 절차                    | 설명                                                              |
| ----------------------- | ----------------------------------------------------------------- |
| 1. 프로젝트 기획        | 프로젝트 목표, 범위, 일정, 예산 등을 정의하고 팀 구성             |
| 2. 웹사이트 기획        | 사이트 목적, 타겟 사용자, 정보구조(IA), 내비게이션 구조 등을 계획 |
| 3. UI/UX디자인          | 와이어프레임, 목업, 프로토타입 제작 및 시각 디자인 완성           |
| 4. 개발                 | HTML/CSS/JavaScript 등을 사용한 프론트엔드·백엔드 구현            |
| 5. 테스트 및 배포       | 크로스 브라우저 테스트, QA 수행 후 실서버에 배포                  |
| 6. 유지보수 및 업데이트 | 사이트 운영 중 오류 수정 및 콘텐츠·기능 업데이트                  |

**`웹디자인 프로세스 3단계`**

| 단계                            | 설명                                                               |
| ------------------------------- | ------------------------------------------------------------------ |
| 사전 제작 단계(Pre-production)  | 프로젝트 기획, 사이트 기획, UI/UX 디자인 등 제작 전 모든 준비 단계 |
| 제작 단계(Production)           | 실제 디자인 및 개발 작업을 수행하는 단계                           |
| 후반 제작 단계(Post-production) | 테스트, 배포, 유지보수 등 제작 완료 후 운영 단계                   |

### 컴퓨터 그래픽스

컴퓨터 그래픽스란 컴퓨터 장치(HW/SW)를 이용하여 입력된 정보를 시각적 요소로 재가공하여 변환한 후 출력하는 일련의 작업과 기술을 말한다.

### 화면 및 기능 요소

**`내비게이션 메뉴(Navigation Menu)`**

내비게이션 메뉴는 가장 많이 클릭되는 요소중 하나로, 단순하고 명확하게 설계하고 구현해야 한다.

내비게이션 메뉴는 사용자가 쉽게 검색하고 효율적으로 이동할 수 있도록 구현한다.

일반적으로 사이트의 상단 또는 측면에 위치시킨다.

**`인터랙티브(Interactive) 요소`**

인터랙티브 요소는 사용자와 웹사이트 간의 상호작용을 촉진하는 기능을 포함하는 요소다.

| 요소                          | 설명                                                                                  |
| ----------------------------- | ------------------------------------------------------------------------------------- |
| 폼(Form)                      | 사용자 입력을 받는 UI 요소. 텍스트 입력, 체크박스, 라디오 버튼, 제출 버튼 등으로 구성 |
| 애니메이션                    | CSS·JavaScript로 UI 요소에 움직임 효과를 적용하여 사용자 경험을 풍부하게 하는 요소    |
| 드래그 앤 드롭(Drag and Drop) | 요소를 클릭한 채 이동시켜 다른 위치에 놓을 수 있는 인터랙션                           |
| Modal Window                  | 현재 페이지 위에 오버레이로 표시되는 팝업 창. 사용자 확인이나 추가 정보 입력에 사용   |
| 피드백(Feedback)              | 사용자 행동에 대한 시스템 반응. 로딩 스피너, 성공/오류 메시지 등                      |

### 웹 트렌드(Web trend)

웹 트렌드는 최신 인터넷 사용자 행동 및 기술의 변화, 기술과 디자인 분야의 현재와 미래 동향을 반영한다.

트렌드는 지속기간에 따라 구분된다.

**`지속기간에 따른 트렌드`**

| 종류                        | 설명                                                                                       |
| --------------------------- | ------------------------------------------------------------------------------------------ |
| 마이크로트렌드(Micro trend) | 단기간(1~2년) 지속되는 트렌드. 특정 집단에서 주로 나타나며 빠르게 변화                     |
| 트렌드(trend)               | 중기간(3~5년) 지속되는 트렌드. 대중적으로 확산되어 사회 전반에 영향을 미침                 |
| 메가트렌드(Mega trend)      | 장기간(10년 이상) 지속되는 거시적 트렌드. 사회·경제·기술 전반에 걸쳐 큰 변화를 이끄는 흐름 |

**`최신 웹 트렌드`**

| 종류                                 | 설명                                                                                      |
| ------------------------------------ | ----------------------------------------------------------------------------------------- |
| 반응형 디자인                        | 다양한 디바이스와 화면 크기에 맞게 레이아웃을 자동으로 최적화하는 디자인 트렌드           |
| AI 기반 맞춤형 경험                  | AI를 활용하여 사용자 행동과 선호도를 분석해 개인화된 콘텐츠와 UI를 제공하는 트렌드        |
| 마이크로 인터랙션(Micro interaction) | 버튼 클릭, 좋아요, 스와이프 등 작은 인터랙션에 섬세한 애니메이션 피드백을 제공하는 트렌드 |
| 미니멀리즘(Minimalism)               | 불필요한 요소를 제거하고 핵심 콘텐츠에 집중하는 단순하고 깔끔한 디자인 트렌드             |
| 다크 모드(Dark mode)                 | 어두운 배경과 밝은 텍스트를 사용하여 눈 피로 감소와 배터리 절약을 제공하는 UI 트렌드      |
| VR/AR 활용                           | 가상 현실(VR)·증강 현실(AR)을 웹에 접목하여 몰입형 인터랙티브 경험을 제공하는 트렌드      |
| 접근성 강화                          | 장애 여부나 환경에 관계없이 모든 사용자가 이용할 수 있도록 WCAG 기준을 준수하는 트렌드    |
| 최적화 및 캐싱(Caching)              | 이미지 압축, 코드 최소화, 캐싱 전략 등을 통해 웹사이트 로딩 속도를 향상시키는 트렌드      |
