## 색청

색청(色聽, Colored hearing)은 소리나 음악을 들을 때 본래의 청각적 지각과 동시에 특정 색채나 형태의 시각적 감각을 함께 느끼는 공감각(Synesthesia)의 일종.

예를 들어, 피아노의 높은음에서는 밝은 노란색을, 첼로의 낮은음에서는 어둡고 짙은 파란색을 보는 식.

### 발생하는 원리

인간의 뇌 발달 과정에서 청각을 담당하는 측두엽과 시각을 담당하는 후두엽 사이의 신경 연결이 일반적인 사람들보다 더 강하게 유지되거나 활성화되어 발생한다.

소리 자극이 청각 피질을 넘어 시각 피질까지 동시에 자극하여 소리가 색으로 보이게 된다.

### 증상과 특징

| 특징             | 설명                                                                                                                   |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------- |
| 음의 높이와 색상 | 일반적으로 저주파(낮은 음)에서는 어둡거나 칙칙한 색, 고주파(높은 음)에서는 밝고 화려한 색이 나타나는 경향              |
| 개인차           | 같은 소리를 들어도 개인마다 다르게 느낌. 누군가에게 C장조(다장조)가 흰색으로 보인다면, 다른 사람에게는 노란색으로 보임 |
| 소리의 질감      | 단순히 단색만 보이는 것이 아니라 선, 도형, 오로라 형태 등 입체적인 질감으로 나타날 수 있음                             |

## 관용색명 / 일반색명

### 관용색명 (Traditional / Conventional Color Name)

옛날부터 전해 내려오거나, 일상에서 흔히 접하는 사물·자연 등의 이름을 따서 부르는 직관적인 색명.

색을 직관적으로 연상하기 쉽고 친근하지만, 사람마다 떠올리는 색의 정확한 밝기나 선명도가 다를 수 있어 표준화가 어렵다.

**`하위 분류 및 예시`**

| 분류           | 설명                                                |
| -------------- | --------------------------------------------------- |
| 식물           | 살구색, 밤색, 개나리색, 라벤더색, 올리브색          |
| 동물           | 쥐색, 살몬색(연어), 카나리아색, 피코크 블루(공작새) |
| 광물/보석      | 금색, 은색, 에메랄드색, 루비색, 황토색              |
| 자연 현상/기타 | 하늘색, 바다색, 초콜릿색, 와인색, 커피색            |

### 일반색명(계통색명, Systematic Color Name)

색의 3속성인 색상, 명도, 채도를 기준으로 규칙에 따라 체계적으로 표현하는 색명.

공식 명칭은 **계통색명** 이라고 한다.

언어만 보고도 누구나 정확한 색의 성격을 유추할 수 있어 디자인, 산업, 표준 규격에서 주로 사용한다.

톤(수식어) + 색상의 형태로 결합하여 사용한다.

**`용어 표현 방식`**

| 구분       | 설명                                                     |
| ---------- | -------------------------------------------------------- |
| 수식어(톤) | 밝은, 어두운, 진한, 연한, 흐린 등                        |
| 기본 색상  | 빨강, 노랑, 파랑, 초록 등                                |
| 표현       | '밝은 노랑', '어두운 회색', '선명한 빨강', '연한 파란색' |

## 디더링

디더링(Dithering)은 제한된 색상으로 더 많은 색상이 있는 것처럼 시각적 착시를 만드는 기법이다.

주로 하위비트 컬러모드에서 사용되며, 색상이 끊겨보이는 밴딩(Banding)현상을 방지하기 위해 사용된다.

단순히 색을 뭉개는 대신, 팔레트에 있는 근접 색상의 픽셀들을 패턴으로 배치하거나 밀도를 조절하여 멀리서 봤을 때 중간색이나 부드러운 명암이 섞인 것처럼 보이게 만드는 원리다.

**`오디오 디더링`**

이미지뿐만 아니라 오디오 분야에서도 디더링이 사용된다.

오디오 비트 심도를 낮추게 되면 그만큼 양자화 노이즈가 발생하는데, 이를 줄이기 위해 일부러 디더링 노이즈를 삽입한다. 이 과정을 통해 역설적으로 노이즈가 감소하여 음질이 높아진다.
