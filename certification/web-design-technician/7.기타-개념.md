## 색청

색청(色聽, Colored hearing)은 소리나 음악을 들을 때 본래의 청각적 지각과 동시에 특정 색채나 형태의 시각적 감각을 함께 느끼는 공감각(Synesthesia)의 일종.

예를 들어, 피아노의 높은음에서는 밝은 노란색을, 첼로의 낮은음에서는 어둡고 짙은 파란색을 보는 식.

### 발생하는 원리

인간의 뇌 발달 과정에서 청각을 담당하는 측두엽과 시각을 담당하는 후두엽 사이의 신경 연결이 일반적인 사람들보다 더 강하게 유지되거나 활성화되어 발생한다.

소리 자극이 청각 피질을 넘어 시각 피질까지 동시에 자극하여 소리가 색으로 보이게 된다.

### 증상과 특징

| 특징             | 설명                                                                                                                   |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------- |
| 음의 높이와 색상 | 일반적으로 저주파(낮은 음)에서는 어둡거나 칙칙한 색, 고주파(높은 음)에서는 밝고 화려한 색이 나타나는 경향              |
| 개인차           | 같은 소리를 들어도 개인마다 다르게 느낌. 누군가에게 C장조(다장조)가 흰색으로 보인다면, 다른 사람에게는 노란색으로 보임 |
| 소리의 질감      | 단순히 단색만 보이는 것이 아니라 선, 도형, 오로라 형태 등 입체적인 질감으로 나타날 수 있음                             |
