## 색채 계획

### 색채 계획 / 색채 계획서

색채 계획은 웹사이트의 목적과 용도에 맞는 요소를 기반으로 하여 색상을 체계적으로 선정하고, 기능적이고 심미적인 배색 조합을 결정하는 방법이다.

색채 계획을 위해 색채 계획서를 작성한다.

색채 계획서 작성 과정중 색채 콘셉트 결정 시 주조색, 보조색, 강조색을 결정한다.

### 주조색 / 보조색 / 강조색

**`주조색(Dominant Color)`**

주조색이란 웹사이트 디자인에서 배색의 기본이 되는 색상으로, 웹사이트의 70~75% 정도를 차지하는 색상이다.

주조색은 주로 웹사이트의 로고, 배경, 주요 메뉴나 버튼에 사용된다.

브랜드를 시각적으로 나타내며, BI(Brand Identity)를 강화하는 효과를 준다.

**`보조색(Secondary Color)`**

보조색은 웹사이트의 약 20~25%를 차지하며, 주조색을 보완하여 디자인의 조화 및 균형을 이루기 위해 사용된다.

**`강조색(Accent Color)`**

강조색은 웹사이트의 약 5~10%를 차지하며, 중요한 요소를 강조하기 위해 사용된다.

주로 주의를 끌기 위한 버튼, 경고 메시지, 중요 링크 등에 포함된다.

## 색채 조화와 색채 조화론

### 색채 조화

색채 조화는 여러 색상이 조화롭게 어울리는 것을 의미하며, 배색 방식에 따라 다른 효과를 나타낸다. 각 색채는 감정적, 심리적, 문화적 등 의미를 포함한다.

**`색채 조화 종류`**

| 종류                      | 설명                                                                     |
| ------------------------- | ------------------------------------------------------------------------ |
| 유사색 조화               | 색상환에서 인접한 유사색끼리 조합. 부드럽고 자연스러운 느낌              |
| 등간격 3색 조화           | 색상환을 3등분(120° 간격)한 3가지 색 조합. 균형감 있으면서 다채로운 효과 |
| 보색 조화                 | 색상환에서 정반대에 위치한 보색 조합. 강한 대비와 생동감 효과            |
| 근접보색 조화             | 보색의 양쪽 인접색 조합. 보색 조화보다 대비가 부드러움                   |
| 톤온톤(Tone-on-Tone) 조화 | 동일 색상에서 톤(명도·채도)만 변화를 주는 조합. 통일감과 세련된 느낌     |
| 명도에 따른 조화          | 명도 차이를 기준으로 단계적으로 색을 조합하는 방법                       |
| 주조색의 조화             | 주조색을 중심으로 보조색·강조색을 조합하여 전체 배색에 통일감 부여       |

### 색채 조화론

미국 색채학자 D. B. Judd는 색채 조화의 심리적 측면을 중시한 색채 조화론을 제시했다.

저드의 색채조화 원리는 네 원리로 구분되며, **대비의 원리** 까지 추가하면 색채 조화의 공통 원리가 된다.

**`색채 조화의 공통 원리`**

| 종류                    | 설명                                                            |
| ----------------------- | --------------------------------------------------------------- |
| 질서의 원리             | 색상·명도·채도가 규칙적·체계적으로 선택된 색끼리 잘 조화된다    |
| 유사의 원리             | 공통 요소(색상·명도·채도 중 하나)를 공유하는 색끼리 잘 조화된다 |
| 친근감(동류)의 원리     | 자연이나 일상에서 익숙하게 접하는 색끼리 잘 조화된다            |
| 명료성(비모호성)의 원리 | 명확하고 분명한 대비 관계에 있는 색끼리 잘 조화된다             |
| 대비의 원리             | 명도·채도·색상에서 뚜렷한 대비를 이루는 색끼리 잘 조화된다      |

**`슈브뢸의 색채 조화론`**

프랑스 화학자 슈브뢸(M. E. Chevreul)은 직물의 날실과 씨실을 통해 색채 조화와 대비에 대한 법칙을 발견하였다.

슈브뢸은 색의 유사성과 대조에서 색채 조화가 이루어진다고 주장했다.

| 종류             | 설명                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------- |
| 동시대비의 원리  | 인접한 두 색이 병치되어 하나의 색상처럼 보이는 현상                                   |
| Dominant Color   | 주도색. 배색에서 지배적인 색을 두어 전체 배색에 통일감을 부여하는 원리                |
| Separation Color | 분리색. 두 색 사이에 무채색(흰·회·검)을 삽입하여 색채를 분리하고 명료성을 높이는 방법 |
| 보색배색의 조화  | 보색 관계의 두 색상의 대비적인 조화가 더 잘이루어진다는 원리                          |

## 배색 조합

### 배색 조합 개념

배색 조합이란 색을 목적에 맞게 표현하기 위해 주변의 색을 고려하여 조화롭게 배치하는 것으로, 배색이라고도 한다.

**`배색에서 고려해야 하는 요소`**

| 요소                         | 설명                                                                                                          |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------- |
| 목적                         | 정보 전달·감성 유발·브랜드 강화 등 배색의 사용 목적에 맞게 색채를 계획해야 한다                               |
| 적은 색상 수                 | 사용 색상 수를 최소화할수록 디자인이 명확하고 통일감 있게 완성된다                                            |
| 주조색 우선 결정             | 전체 배색의 기준이 되는 주조색을 먼저 정한 후, 보조색·강조색 순서로 결정한다                                  |
| 3원색 고려                   | 색의 기본인 3원색(빨강·노랑·파랑)을 기준으로 배색을 구성하는 것이 효과적이다                                  |
| 감정, 지각적 효과 고려       | 색이 인간에게 주는 감정적·심리적 반응을 고려하여 배색한다                                                     |
| 면적의 비례와 대비 효과 고려 | 색의 사용 면적 비율이 시각적 인상에 크게 영향을 미치므로, 주조·보조·강조색의 비율과 대비 효과를 함께 고려한다 |

### 색상에 의한 배색

색상(Color)을 중심으로 배색을 결정하는 방식이다.

**`색상 배색 효과`**

| 배색     | 효과                            | 배색 예시                |
| -------- | ------------------------------- | ------------------------ |
| 동일색상 | 통일감, 차분하고 안정적인 느낌  | 연파랑 + 파랑 + 진파랑   |
| 유사색상 | 부드럽고 자연스러운 조화감      | 노랑 + 연두 + 초록       |
| 보색색상 | 강한 대비와 생동감, 높은 주목성 | 빨강 + 초록, 파랑 + 주황 |
| 반대색상 | 뚜렷하고 개성 있는 대비 효과    | 빨강 + 파랑, 노랑 + 보라 |

### 명도에 의한 배색

색의 밝기(Lightness)를 중심으로 배색을 결정하는 방식이다.

**`명도 배색 효과`**

| 배색     | 효과                              | 배색 예시                         |
| -------- | --------------------------------- | --------------------------------- |
| 유사명도 | 부드럽고 온화한 느낌, 안정감      | 연노랑 + 연베이지 (밝은 계열끼리) |
| 반대명도 | 강렬하고 명확한 대비, 주목성 증가 | 흰색 + 검정, 밝은 색 + 어두운 색  |

### 채도에 의한 배색

색의 채도(Saturation)를 중심으로 배색을 결정하는 방식이다.

**`채도 배색 효과`**

| 배색     | 효과                      | 배색 예시                       |
| -------- | ------------------------- | ------------------------------- |
| 유사채도 | 부드럽고 통일감 있는 느낌 | 고채도끼리 또는 저채도끼리 조합 |
| 반대채도 | 명확한 대비, 강조 효과    | 선명한 빨강 + 무채색(회색)      |

### 색의 대비

색의 대비란 함께 배치된 색이 서로 영향을 받아 본래의 색보다 더욱 강조되는 효과를 말한다.

**`동시대비(Simultaneous Contrast)`**

인접되어 있거나 다른 색 안에 놓여있는 두 가지 색을 동시에 서로 영향을 받아 색상이 다르게 보이는 현상.

| 종류                                   | 설명                                                                                           |
| -------------------------------------- | ---------------------------------------------------------------------------------------------- |
| 색상대비(Hue contrast)                 | 색상이 다른 두 색이 인접할 때 서로의 색상 차이가 더욱 크게 느껴지는 현상                       |
| 명도대비(Value / Lightness contrast)   | 명도가 다른 두 색이 인접할 때 밝은 색은 더 밝게, 어두운 색은 더 어둡게 보이는 현상             |
| 채도대비(Chroma / Saturation contrast) | 채도가 다른 두 색이 인접할 때 고채도 옆 색은 더 탁하게, 저채도 옆 색은 더 선명하게 보이는 현상 |
| 보색대비(Complementary color contrast) | 보색 관계의 두 색이 인접할 때 서로의 색이 더욱 선명하고 강렬하게 보이는 현상                   |
| 연변대비(Edge contrast)                | 두 색이 인접하는 경계 부분에서 대비 효과가 더욱 강하게 나타나는 현상                           |

**`계시대비(Successive contrast)`**

색상을 일정 시간동안 보고난 후 다른 색상을 봤을 때 발생되는 소극적 잔상 효과로, 계속대비라고도 한다.

**`한난대비(Warm-cool contrast)`**

차가운 색과 따뜻한 색을 함께 배치할 경우 강조되어 더욱 강하게 느껴지는 현상

**`면적대비(Area contrast)`**

면적의 크기에 따라 색이 다르게 느껴지는 현상.

- 면적이 큰 도형은 작은 도형보다 명도와 채도가 더욱 높게 느껴지는 효과가 있다.

### 색의 지각적 효과

**`동화현상(Assimilation Effect)`**

인접한 색의 영향을 받아 본래의 색이 주변 색 방향으로 가깝게 느껴지는 현상. 대비 현상과 반대 방향으로 작용한다.

- 예: 회색이 빨간 줄무늬 사이에 놓이면 붉게 느껴짐

**`잔상효과(Afterimage Effect)`**

특정 색을 일정 시간 응시한 후 시선을 옮겼을 때 원래 색의 보색이나 명도가 반전된 상이 보이는 현상.

| 종류      | 설명                                            |
| --------- | ----------------------------------------------- |
| 양성 잔상 | 원래 색과 동일한 색이 잠시 지속되어 보이는 현상 |
| 음성 잔상 | 원래 색의 보색이 보이는 현상 (가장 일반적)      |

**`주목성 / 명시성`**

| 종류               | 설명                                                                             | 예시                    |
| ------------------ | -------------------------------------------------------------------------------- | ----------------------- |
| 주목성(Notice)     | 색이 사람의 시선을 끄는 정도. 난색·고채도·고명도일수록 주목성이 높다             | 경고 표지판의 빨강·노랑 |
| 명시성(Visibility) | 색이 배경에서 얼마나 잘 보이는지의 정도. 배경과의 명도 차가 클수록 명시성이 높다 | 흰 배경 + 검정 글씨     |

**`진출색 / 후퇴색(Advancing / Receding Color)`**

| 종류   | 특성                                                   | 예시             |
| ------ | ------------------------------------------------------ | ---------------- |
| 진출색 | 실제보다 앞으로 튀어나와 보이는 색. 난색·고명도·고채도 | 빨강, 주황, 노랑 |
| 후퇴색 | 실제보다 뒤로 물러나 보이는 색. 한색·저명도·저채도     | 파랑, 청록, 남색 |

**`팽창색 / 수축색(Expansive / Contractive Color)`**

| 종류   | 특성                                          | 예시              |
| ------ | --------------------------------------------- | ----------------- |
| 팽창색 | 실제보다 크게 느껴지는 색. 고명도·난색·고채도 | 흰색, 밝은 노랑   |
| 수축색 | 실제보다 작게 느껴지는 색. 저명도·한색·저채도 | 검정, 어두운 파랑 |

## 색과 색채의 심리적 / 기능적 작용

### 색채의 심리적 작용

**`색채의 연상`**

색채의 연상이란 색을 볼 때 심리적 활동의 영향으로 구체적인 형상, 의미, 관념이 떠오르는 것이다.

| 종류        | 설명                                                         | 예시                             |
| ----------- | ------------------------------------------------------------ | -------------------------------- |
| 구체적 연상 | 색을 보고 실제 사물·자연 현상 등 구체적 대상을 떠올리는 연상 | 파랑 → 하늘·바다, 초록 → 나무·풀 |
| 추상적 연상 | 색을 보고 감정·개념·분위기 등 추상적인 것을 떠올리는 연상    | 빨강 → 열정, 파랑 → 차가움       |

**`색채의 상징`**

색채의 상징이란 연상으로 나타난 반응을 사회, 문화적인 상징으로 사용한 것이다.

| 구분        | 설명                                                                             |
| ----------- | -------------------------------------------------------------------------------- |
| 전통과 역사 | 문화권에서 오랫동안 전해 내려온 색의 의미. 예: 한국 오방색(빨·파·노·흰·검)       |
| 환경과 기후 | 자연환경·기후에 따라 색채 선호와 상징이 달라짐. 예: 열대 - 원색, 북유럽 - 무채색 |
| 연령        | 연령대에 따라 선호 색이 다름. 예: 어린이 - 원색, 노년층 - 차분한 중간 톤         |
| 신분        | 과거에는 색이 사회적 지위를 나타냄. 예: 보라 - 황제·귀족, 노랑 - 중국 황실       |
| 지역        | 나라·지역마다 같은 색의 상징이 다름. 예: 흰색 - 서양은 순수, 동양 일부는 상복    |
| 기업        | 브랜드 아이덴티티를 위해 특정 색을 전략적으로 사용. 예: 코카콜라 - 빨강          |
| 종교        | 종교마다 신성하거나 금기시하는 색이 있음. 예: 기독교 - 흰색, 이슬람 - 초록       |

**`공감각(Synesthesia)`**

공감각이란 한 가지 감각이 자극될 때 다른 감각이 동시에 반응하는 현상을 의미한다.

색채는 시각적인 효과 외에 다양한 공감각을 느끼게 한다.

| 구분 | 설명                                          | 예시                                           |
| ---- | --------------------------------------------- | ---------------------------------------------- |
| 촉각 | 색을 보고 따뜻함·차가움·거칠기 등 촉감을 느낌 | 빨강·주황 → 따뜻함, 파랑 → 차가움              |
| 미각 | 색을 보고 단맛·신맛·쓴맛 등 맛을 연상함       | 빨강·주황 → 단맛, 노랑 → 신맛, 초록 → 쓴맛     |
| 후각 | 색을 보고 특정 향기나 냄새를 연상함           | 보라 → 라벤더 향, 초록 → 풀냄새                |
| 청각 | 색을 보고 음의 높낮이나 크기 등 소리를 연상함 | 밝은 색 → 고음·경쾌함, 어두운 색 → 저음·무거움 |
